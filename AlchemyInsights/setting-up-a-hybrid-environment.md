# <a name="setting-up-a-hybrid-environment"></a>Configurarea unui mediu hibrid

Pentru a permite utilizatorilor să lucreze fără probleme, indiferent dacă datele sunt stocate pe serverele locale sau în Microsoft 365, poate fi necesar să configurați un mediu hibrid. Pentru a obține o prezentare generală a opțiunilor dvs., consultați [Integrarea Microsoft 365 cu medii locale](https://docs.microsoft.com/office365/enterprise/office-365-integration).
