# <a name="submit-microsoft-cloud-for-healthcare-support-requests"></a>Trimiteți solicitări de asistență Microsoft Cloud pentru asistență medicală

Clientul Microsoft Cloud pentru asistență medicală vă rugăm să trimiteți biletul prin noul [Centru de administrare Power platform.](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)

Portalul nou oferă o cale mai directă către asistența asistată optimizată pentru Microsoft Cloud pentru asistență medicală.
