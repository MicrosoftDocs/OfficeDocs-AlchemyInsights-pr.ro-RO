# <a name="use-microsoft-edge-extensions"></a>Utilizarea extensiilor Microsoft Edge

[Extensiile](https://go.microsoft.com/fwlink/?linkid=2135619) sunt Add-ons pe care le puteți instala în Microsoft Edge pentru a bloca reclamele, a gestiona parole, a traduce text și a vă personaliza experiența de navigare în multe alte moduri.
