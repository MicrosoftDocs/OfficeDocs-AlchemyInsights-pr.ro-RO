# <a name="use-microsoft-graph-api-to-manage-groups"></a>Utilizarea API-ului Microsoft Graph pentru gestionarea grupurilor

Grupurile sunt colecții de utilizatori și alți directori care partajează accesul la resurse în Microsoft Services sau în aplicația dvs. Microsoft Graph oferă API-uri pe care le puteți utiliza pentru a crea și a gestiona diverse tipuri de grupuri și funcționalități de grup, în conformitate cu scenariul. Toate operațiunile legate de grup în Microsoft Graph necesită consimțământul administratorului. Pentru mai multe informații, consultați [lucrul cu grupurile în Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview).
