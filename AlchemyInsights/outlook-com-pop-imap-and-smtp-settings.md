# <a name="pop-imap-and-smtp-settings-for-outlookcom"></a>Setări POP, IMAP și SMTP pentru Outlook.com

Dacă doriți să adăugați contul Outlook.com la o aplicație de e-mail care acceptă POP sau IMAP, iată setările serverului de care veți avea nevoie:
  
- **Nume server IMAP:** outlook.office365.com
- **Port IMAP:** 993
- **Metoda de criptare IMAP:** Tls
- **Pop nume server:** outlook.office365.com  
- **Port POP:** 995  
- **Metoda de criptare POP:** Tls  
- **Nume server SMTP:** smtp.office365.com
- **Portul SMTP:** 587
- **Metoda de criptare SMTP:** Starttls

Aflați mai multe despre [adăugarea contului Outlook.com la o altă aplicație de e-mail sau la un alt dispozitiv inteligent](https://support.office.com/article/73f3b178-0009-41ae-aab1-87b80fa94970?wt.mc_id=Office_Outlook_com_Alchemy).
