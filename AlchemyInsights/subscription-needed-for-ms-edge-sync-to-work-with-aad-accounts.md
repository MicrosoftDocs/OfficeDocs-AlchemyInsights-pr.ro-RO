# <a name="subscription-needed-for-microsoft-edge-sync-to-work-with-azure-active-directory-accounts"></a>Abonamentul necesar pentru Microsoft Edge Sync pentru a funcționa cu conturi Azure Active Directory

Sincronizarea va funcționa cu oricare dintre următoarele abonamente:

1. Azure Active Directory Premium (P1 și P2)
1. Microsoft 365 Business Premium
1. Office 365 E3 și versiunile de mai sus
1. Azure Information Protection (P1 și P2)
1. Toate abonamentele Office 365 Education:
    - Aplicații Microsoft pentru studenți sau profesori
    - Exchange Online pentru studenți sau profesori
    - Office 365 a1 sau superior
    - Microsoft 365 a1 sau superior
    - Azure Information Protection P1 sau P2, pentru studenți sau profesori
