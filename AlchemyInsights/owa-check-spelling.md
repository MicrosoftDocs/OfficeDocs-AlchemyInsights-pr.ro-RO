# <a name="use-your-browser-to-check-spelling"></a>Utilizarea browserului pentru a verifica ortografia

Nu există nicio opțiune de verificare ortografică în Outlook pe web. Pentru a vă verifica ortografia, va trebui să utilizați browserul web. Verificarea ortografică este disponibilă în Microsoft Edge, Internet Explorer 10 și versiuni mai recente și versiunile curente de Firefox, Chrome și Safari. Verificați opțiunile pentru browserul web pentru a afla mai multe despre cum să verificați ortografia.
