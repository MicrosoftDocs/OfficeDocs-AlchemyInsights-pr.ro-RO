# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365 URL-uri și intervale de adrese IP

Microsoft 365 necesită conectivitate la Internet prin intermediul unor adrese IP specifice, denumite *puncte finale*.
Pentru o listă de puncte finale care trebuie să fie accesibile, consultați [Microsoft 365 URL-uri și intervale de adrese IP](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Serviciul web Adresă IP și URL Microsoft 365 vă ajută să identificați și să diferențiați mai bine traficul de rețea Microsoft 365, facilitând evaluarea, configurarea și rămânerea la curent cu modificările. Consultați [Serviciul web Adresă IP și URL Microsoft 365](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
