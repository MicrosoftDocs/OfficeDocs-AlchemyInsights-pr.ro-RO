# <a name="how-to-buy-a-copy-of-windows-10"></a>Cum se cumpără o copie a Windows 10

Pentru prețuri actualizate și pentru a cumpăra și a descărca o versiune completă de Windows 10, accesați pagina [de Microsoft Store](https://www.microsoft.com/store/b/windows).

**Notă:** Upgrade Windows 10 gratuit prin aplicația Obțineți Windows 10 (GWX) nu mai este finalizat la 29 iulie 2016.

**Important:** Dacă utilizați Windows 7, Microsoft vă recomandă să treceți la un PC nou care rulează Windows 10, deoarece Windows 7 a ajuns la sfârșitul ciclului de viață al asistenței la 14 ianuarie 2020.
