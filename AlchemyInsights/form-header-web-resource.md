# <a name="form-header---web-resource"></a>Antet formular - resursă Web

Aveți posibilitatea să adăugați sau să editați resurse web într-un formular pentru a face formularul mai atractiv sau mai util pentru utilizatori. Pentru început, asigurați-vă că aveți permisiunile de securitate administrator de sistem sau de particularizare de sistem sau permisiuni echivalente.  

**Notă** Nu aveți posibilitatea să includeți o resursă Web într-un antet sau subsol de formular.

Pentru mai multe informații, consultați [Crearea și editarea unei resurse Web într-un formular](https://docs.microsoft.com/dynamics365/customer-engagement/customize/create-edit-web-resources#create-and-edit-a-web-resource-on-a-form).
