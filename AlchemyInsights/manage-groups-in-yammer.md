# <a name="manage-groups-in-yammer"></a>Gestionaregrupuri în Yammer

- **Permisiunea de creare a grupului.** Yammer nu controlează cine poate crea un grup. Cu toate acestea, este posibil să gestionați crearea de grup pentru grupurile Microsoft 365 conectate. Consultați [Gestionarea persoanelor care pot crea Grupuri Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups) pentru mai multe informații.

- **Calitatea de membru dinamic al grupului.** Consultați [Crearea unui grup dinamic în Yammer](https://docs.microsoft.com/yammer/manage-yammer-groups/create-a-dynamic-group) pentru informații despre crearea de membri dinamici pentru grupurile Microsoft 365 conectate la Yammer.

- **Se restaurează grupurile șterse.** Restaurarea grupurilor Yammer este posibilă prin [Restaurarea unui grup Microsoft 365 șters](https://docs.microsoft.com/microsoft-365/admin/create-groups/restore-deleted-group) grupuri neconectate nu pot fi restaurate după ștergere.

Pentru mai multe informații despre gestionarea unui grup, consultați [Gestionarea unui grup în Yammer](https://support.office.com/article/Manage-a-group-in-Yammer-6e05c6d6-5548-4c88-89cd-e6757a514ef2). Pentru informații despre grupurile Microsoft 365 conectate, consultați [Yammer și Microsoft 365 Groups](https://docs.microsoft.com/yammer/manage-yammer-groups/yammer-and-office-365-groups).
