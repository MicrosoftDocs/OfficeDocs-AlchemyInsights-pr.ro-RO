# <a name="public-folder-access-from-mobile-devices"></a>Folderul public acces de pe dispozitive mobile

Foldere publice nu sunt accesibile de pe dispozitive mobile.
  
Dacă doriţi să aibă funcţionalitate similară cu folderele publice şi de a accesa de pe dispozitive mobile, consultaţi [Aflaţi despre Office 365 grupuri](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) pentru o alternativă mai bună.
