# <a name="publishing-infrastructure-with-a-communication-site"></a>Infrastructura publicare cu un Site de comunicare

Activarea caracteristicii infrastructurii de publicare SharePoint Server este **neacceptat** cu site-uri de comunicare. Vă rugăm să oferiți feedback-ul dvs., în [UserVoice](https://go.microsoft.com/fwlink/?linkid=2047322&amp;clcid=0x409). 
  
Vedea [activa funcţiile de publicare](https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518) pentru mai multe informaţii.
