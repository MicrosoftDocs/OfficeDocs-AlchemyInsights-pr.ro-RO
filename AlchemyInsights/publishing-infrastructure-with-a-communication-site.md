# <a name="publishing-infrastructure-with-a-communication-site"></a>Infrastructura publicare cu un Site de comunicare


Activarea caracteristicii infrastructurii de publicare SharePoint Server este **neacceptat** cu site-uri de comunicare. 
  
Pentru mai multe informaţii, consultaţi [Mutarea la publicarea site-uri de site-uri de comunicare](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
