# <a name="change-your-pin"></a>Schimba codul PIN

Outlook.com nu foloseste un ac, dar s-ar putea conecta la PC-ul cu un ac sau Windows Hello. Pentru a schimba PIN-ul la Windows 10:

1. Selectaţi butonul Start sau apăsaţi tasta Windows de pe tastatură, apoi selectaţi **Setări** > **Conturi** > **Opţiuni de conectare**.
2. Selectaţi butonul **modificare** sub **codul PIN**.
3. Introduceţi codul PIN curent, apoi introduceţi şi confirmaţi dumneavoastră PIN nou dedesubt.

[Citeşte mai multe despre Windows Hello.](https://support.microsoft.com/help/17215/)
