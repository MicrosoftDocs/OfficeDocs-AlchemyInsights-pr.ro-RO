# <a name="onedrive-recycle-bin"></a>Coșul de reciclare OneDrive

Atunci când ștergeți un element sau un site din OneDrive, acesta nu este eliminat imediat. Elementele șterse intră în coșul de reciclare pentru un interval de timp. În acest timp, puteți restaura elementele pe care le-ați șters în locația lor originală. [Aflați mai multe](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f?ui=en-US&rs=en-US&ad=US).
