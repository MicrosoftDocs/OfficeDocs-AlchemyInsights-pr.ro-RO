# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Utilizarea unei cutii poștale partajate pe un dispozitiv mobil

Pentru a utiliza o cutie poștală partajată pe dispozitivul mobil, ar trebui să [Descărcați Outlook Mobile](https://products.office.com/outlook-mobile-for-android-and-ios) și apoi [Adăugați o cutie poștală partajată în Outlook Mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Pentru a obține ajutor: [Obțineți ajutor în aplicație pentru Outlook pentru iOS și Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
