# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Utilizarea unei cutii poștale partajate pe un dispozitiv mobil

Pentru a utiliza o cutie poștală partajată pe dispozitivul mobil, trebuie să [descărcați Outlook mobile](https://products.office.com/outlook-mobile-for-android-and-ios) și apoi [să adăugați o cutie poștală partajată la Outlook mobile](https://support.office.com/article/Add-a-shared-mailbox-to-Outlook-mobile-f866242c-81b2-472e-8776-6c49c5473c9f).
  
Pentru a obține ajutor: [obțineți ajutor în aplicație pentru Outlook pentru iOS și Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
