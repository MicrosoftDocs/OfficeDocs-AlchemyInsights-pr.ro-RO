# <a name="activating-windows-10-subscriptions"></a>Activarea abonamentelor Windows 10

Microsoft 365 Business, Microsoft 365 Education, Microsoft 365 și Enterprise furnizează licențe digitale pentru Windows 10 ca parte a abonamentului. Pentru a activa abonamentul, dispozitivul trebuie să fi activat deja Windows 10 Pro (1703 sau o versiune ulterioară). Pentru a vedea o listă completă de cerințe pentru activarea unui abonament Windows 10, consultați [cerințe de activare a abonamentului Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
