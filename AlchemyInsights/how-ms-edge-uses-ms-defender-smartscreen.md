# <a name="how-microsoft-edge-uses-microsoft-defender-smartscreen"></a>Cum utilizează Microsoft Edge Microsoft Defender SmartScreen

Pentru a vă menține în siguranță pe web, Microsoft Edge utilizează Microsoft Defender SmartScreen ca sistem de avertizare timpurie împotriva site-urilor web care comit atacuri de phishing și distribuie programe malware prin atacuri prioritare.

Administratorii pot configura Microsoft Defender SmartScreen utilizând politica de grup, Microsoft Intune sau setările de gestionare a dispozitivelor mobile (MDM). În funcție de modul în care configurați Microsoft Defender SmartScreen, puteți să le afișați utilizatorilor o pagină de avertisment și să le permiteți să continue la site sau să blocheze site-ul în întregime.

Pentru a afla mai multe, consultați [asistența Microsoft Edge pentru Microsoft Defender SmartScreen](https://go.microsoft.com/fwlink/?linkid=2133081) și [Garda de aplicație Microsoft Defender](https://go.microsoft.com/fwlink/?linkid=2132839).
