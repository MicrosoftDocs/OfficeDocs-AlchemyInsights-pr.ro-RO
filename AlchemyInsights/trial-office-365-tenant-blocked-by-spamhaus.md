# <a name="error-when-a-microsoft-365-trial-user-sends-email-client-host-blocked-using-spamhaus"></a>Eroare atunci când un utilizator de încercare Microsoft 365 trimite e-mail: Client gazdă blocat utilizând Spamhaus

Se pare că trimiteți e-mailuri de la o nouă organizație de încercare Microsoft 365. Pentru a preveni abuzurile, blocăm adresele IP despre care se știe că au fost compromise anterior. Această verificare specifică nu mai este impusă după ce începeți să utilizați licențe plătite. Este posibil să puteți obține ajutor temporar solicitând o nouă adresă IP de la furnizorul de servicii internet (ISP) sau prin eliminarea blocului de pe site-ul Web [Spamhaus.](https://go.microsoft.com/fwlink/p/?linkid=123245)
