# <a name="perform-access-reviews-in-azure-active-directory"></a>Efectuarea revizuirilor Access în Azure Active Directory

Recenzenții desemnați pot efectua revizuiri Azure Active Directory Access pentru apartenențele la grup, accesul la aplicații și atribuirile de roluri. Aceste revizuiri ajută la reducerea riscurilor de securitate pe care le poate face organizația atunci când utilizatorii au mai mult acces decât au nevoie. Puteți efectua revizuiri Access utilizând [aplicațiile mele](https://go.microsoft.com/fwlink/?linkid=2134605) sau [accesul meu](https://go.microsoft.com/fwlink/?linkid=2134505).
