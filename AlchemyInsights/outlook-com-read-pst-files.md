# <a name="pst-data-files-in-outlookcom"></a>PST fişiere de date în Outlook.com

Nu este posibil să o Outlook de import. PST fişier de date la contul Outlook.com utilizând un browser.

Pentru a importa un. PST fişier de date, trebuie să utilizaţi Outlook desktop app. Pentru a afla mai multe, citiţi [Import de e-mail, contactele și calendarul la un .pst Outlook.](https://support.office.com/article/431a8e9a-f99f-4d5f-ae48-ded54b3440ac?wt.mc_id=Office_Outlook_com_Alchemy)
