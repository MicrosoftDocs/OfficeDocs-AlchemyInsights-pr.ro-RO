# <a name="webpage-html-web-resources-limitations"></a>Limitări de resurse web pagină Web (HTML)

* O resursă web HTML nu poate conține niciun cod care trebuie executat pe server. Paginile ASP.NET nu pot fi încărcate ca resurse web HTML.

* Resursele web HTML pot accepta doar un număr limitat de parametri de șir de interogare. Mai multe informații: [treceți parametrii la resurse web HTML](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
