# <a name="enable-versioning-for-a-list-or-library"></a>Activarea controlului versiunilor pentru o listă sau o bibliotecă

1. Accesați lista sau biblioteca.
    
2. Faceți clic pe **Setări**, apoi faceți clic pe Setări **bibliotecă** sau pe **Setări listă**.
    
3. Faceți clic pe **Setări versiune**.
    
4. Selectați setările dorite, apoi faceți clic pe **OK**.
