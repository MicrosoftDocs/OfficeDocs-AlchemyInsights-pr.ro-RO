# <a name="enable-send-immediately-when-connected"></a>Activați trimiterea imediat atunci când sunteți conectat
 
1. Pe fila fișier, faceți clic pe **Opțiuni**.

2. În caseta de dialog Opțiuni Outlook, faceți clic pe **Complex**.

3. În secțiunea trimitere și primire, faceți clic pentru a activa **trimiterea imediat atunci când sunteți conectat**. Faceți clic pe **OK**.
