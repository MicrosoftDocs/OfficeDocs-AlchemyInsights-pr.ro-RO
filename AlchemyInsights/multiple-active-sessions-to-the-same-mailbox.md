# <a name="multiple-active-sessions-to-the-same-mailbox"></a>Mai multe sesiuni active în aceeași cutie poștală

Pentru a controla utilizarea resurselor Exchange, o cutie poștală are un "buget".

Excepția suprabugetare poate fi declanșată de următoarele circumstanțe, dar nu se limitează la acestea:

- Câteva file de browser sunt deschise în cadrul aceleiași sesiuni Outlook Web App.

- Câteva sesiuni Outlook Web App active în aceeași cutie poștală.

- Alte câteva aplicații client (Outlook, Outlook Mobile, o aplicație client terță parte) accesează cutia poștală în același timp.

- Operațiunile de lungă durată, ar fi executarea solicitărilor de căutare, se efectuează dintr-o altă sesiune de cutie poștală activă.
