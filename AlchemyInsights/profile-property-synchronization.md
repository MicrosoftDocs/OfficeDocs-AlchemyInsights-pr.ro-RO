# <a name="profile-property-synchronization"></a>Sincronizarea proprietății profilului

Nu este posibil să mapați proprietăți suplimentare pentru AD Import pentru a sincroniza din Azure Active Directory la aplicația profil utilizator. AD Import este limitat la un set de proprietăți preconfigurate pentru a garanta performanța consecventă a lucrării cronometru.
  
[Aflați mai multe despre proprietățile profilului care se sincronizează](https://go.microsoft.com/fwlink/?linkid=875671)
