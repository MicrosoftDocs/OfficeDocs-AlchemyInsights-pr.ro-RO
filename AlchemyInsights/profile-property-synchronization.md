# <a name="profile-property-synchronization"></a>Sincronizarea proprietății profil

Nu este posibil să mapați proprietăți suplimentare pentru importul de reclame pentru a sincroniza de la Azure Active Directory la aplicația de profil de utilizator. Importul RECLAMElor este limitat la un set preconfigurat de proprietăți pentru a garanta performanța consecventă a activității cronometrare.
  
[Aflați mai multe despre proprietățile de profil care se sincronizează](https://go.microsoft.com/fwlink/?linkid=875671)
