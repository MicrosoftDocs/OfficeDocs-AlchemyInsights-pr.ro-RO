# <a name="profile-property-synchronization"></a>Profil proprietate sincronizare

Nu este posibil la spre plan proprietăţi suplimentare pentru AD Import la spre sync la Azure Active Directory pentru aplicația de profil utilizator. AD Import este limitată la un set preconfigurate de proprietăţi pentru a garanta performanţă consecventă de cronometrare.
