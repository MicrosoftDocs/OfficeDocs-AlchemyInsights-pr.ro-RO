# <a name="profile-property-synchronization"></a>Sincronizarea proprietății profilului

Nu este posibil să se mape proprietăți suplimentare pentru AD import pentru a sincroniza de la Azure Active Directory la aplicația profil utilizator. AD import este limitat la un set preconfigurat de proprietăți pentru a garanta performanța consecventă a cronometrului.
  
[Aflați mai multe despre proprietățile profilului care se sincronizează](https://go.microsoft.com/fwlink/?linkid=875671)
