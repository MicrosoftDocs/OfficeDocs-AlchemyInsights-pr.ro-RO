# <a name="update-billing-info-under-azure-enterprise-agreement"></a>Actualizarea informațiilor de facturare din acordul Azure Enterprise

Nu se acceptă modificarea informațiilor de plată pentru abonamentul Azure Enterprise.

Dacă doriți să modificați adresa de pe factura existentă, vă rugăm să contactați reprezentantul Microsoft.
