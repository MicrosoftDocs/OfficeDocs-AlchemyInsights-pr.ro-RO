# <a name="delete-the-office-document-cache"></a>Şterge Office Document Cache

Dacă aţi primit un prompt Excel care **este disponibilă o versiune mai nouă a fişierului**, încercaţi să ştergeţi Office Document Cache urmând instrucţiunile din [şterge memoria Office Document Cache](https://support.office.com/article/b1d3765e-d71b-4bb8-99ca-acd22c42995d). Apoi încercaţi redeschiderea fişierul.
