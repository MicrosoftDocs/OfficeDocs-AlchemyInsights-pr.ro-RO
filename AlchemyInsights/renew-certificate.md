# <a name="renew-certificate"></a>Reînnoire certificat

Pentru a reînnoi un certificat care va expira în curând, consultați [gestionarea certificatelor pentru sign-on federativ unic în Azure Active Directory](https://docs.microsoft.com/azure/active-directory/manage-apps/manage-certificates-for-federated-single-sign-on#renew-a-certificate-that-will-soon-expire).

Pentru informații suplimentare despre cum să gestionați certificatele de semnare a tokenurilor și să le mențineți sincronizate cu Azure AD, [reînnoiți certificatele de federalizare pentru Microsoft 365 și Azure Active Directory](https://docs.microsoft.com/azure/active-directory/hybrid/how-to-connect-fed-o365-certs).
