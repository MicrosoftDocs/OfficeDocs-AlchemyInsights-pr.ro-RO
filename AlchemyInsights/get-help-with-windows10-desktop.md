# <a name="get-help-with-windows-10-desktop"></a>Obțineți ajutor pentru desktopul Windows 10

- Pentru a crea o comandă rapidă pe desktop, [consultați Fixarea aplicațiilor și folderelor pe desktop sau pe bara de activități](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0).

- Pentru a mări pictogramele sau textul în Windows 10, consultați [Modificarea dimensiunii pictogramelor de pe desktop](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211).

- Pentru a afișa pictogramele de pe desktop în Windows 10, consultați [Afișarea pictogramelor pentru desktop în Windows 10](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20).
