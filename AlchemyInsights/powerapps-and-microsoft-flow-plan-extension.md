# <a name="powerapps-and-microsoft-flow-plan-extension"></a>Extensia planului PowerApps și Microsoft Flow

Dacă intenționați să solicitați o extensie de plan pentru **Microsoft Flow** sau **PowerApps** și entitatea găzduită are deja utilizatori activi ai uneia dintre următoarele caracteristici, nu este necesar să depuneți o solicitare de asistență.

- Crearea și publicarea conectorilor personalizați în PowerApps și Microsoft Flow
- Acțiuni particularizate HTTP integrate în Flow în afara SharePoint și OneDrive
- Integrarea datelor locale prin intermediul gateway-ului de date local

Clienții cu utilizatori activi ai acestor caracteristici vor primi o extensie automată până la 31 ianuarie 2020 sau expirarea termenului de abonament Microsoft 365 existent, oricare dintre acestea este mai lungă.
