# <a name="fix-windows-update-errors"></a>Remedierea erorilor Windows Update

Dacă primiți un cod de eroare în timp ce descărcați și instalați Windows Update, **depanatorul Actualizare** vă poate ajuta să rezolvați problema.

- [Descărcați depanatorul Actualizare pentru Windows 10](https://support.microsoft.com/help/4027322/windows-update-troubleshooter)
- Rularea depanatorului. Când se termină, este cea mai bună practică să reporniți dispozitivul. Apoi căutați actualizări noi, accesând [> setări > actualizare & securitate > Windows Update](ms-settings:windowsupdate).

Dacă încă nu funcționează, consultați [Windows Update: întrebări frecvente](https://support.microsoft.com/help/12373/windows-update-faq) pentru mai multe informații.
