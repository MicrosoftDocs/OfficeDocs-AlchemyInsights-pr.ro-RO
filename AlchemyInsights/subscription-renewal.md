# <a name="subscription-recurring-billing"></a>Facturare recurentă pentru abonament

Dacă doriți să știți când va fi facturat abonamentul lunar sau anual (dacă **facturarea recurentă** este activată) sau expiră (dacă **facturarea recurentă** este dezactivată):
  
1. În centrul de administrare, accesați pagina **Facturare** \> **[produse.](https://go.microsoft.com/fwlink/p/?linkid=842054)**

2. Pentru abonamentul despre care doriți să aflați mai multe, sub **Facturare**, consultați **Reștiri la** data **sau expirarea.**

4. Pentru a efectua o modificare, selectați **editați,** apoi, în panoul Setări **de facturare recurente,** **selectați Dezactivare** pentru a o dezactiva. Dacă facturarea recurentă este deja **dezactivată,** selectați Lunar sau **O dată** pentru a o activa.

Pentru mai multe informații, consultați [Activarea sau dezactivarea facturării recurente](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/renew-your-subscription).
