# <a name="error-onedrive-cannot-connect-to-windows"></a>Eroare: OneDrive nu se poate conecta la Windows

Pentru a rezolva mesajul de eroare, căutați fișierul cldflt.sys pentru a vedea dacă rulează și, dacă nu este, activați-l. 
  
Pentru informații suplimentare despre cum să activați fișierul cldflt.sys, consultați [eroarea OneDrive nu se poate conecta la Windows "atunci când accesați fișiere în OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
