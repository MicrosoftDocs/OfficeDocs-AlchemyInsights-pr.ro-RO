# <a name="error-onedrive-cannot-connect-to-windows"></a>Eroare: OneDrive nu se poate conecta la Windows

Pentru a rezolva mesajul de eroare, căutați fișierul cldflt.sys pentru a vedea dacă se execută și dacă nu este, activați-l. 
  
Pentru informații suplimentare despre se activează fișierul cldflt.sys, consultați [OneDrive nu se poate conecta la Windows eroare la accesarea fișierelor în OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
