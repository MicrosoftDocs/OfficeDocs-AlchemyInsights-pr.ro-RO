# <a name="learn-about-microsoft-365-groups"></a>Aflați despre grupurile Microsoft 365

Microsoft 365 Groups este un serviciu care funcționează cu instrumentele Office 365 pe care le utilizați deja, astfel încât să puteți colabora cu colegii de echipă atunci când scrieți documente, să creați foi de calcul, să lucrați la planuri de proiect, să planificați întâlniri sau să trimiteți mesaje de e-mail. Pentru mai multe informații, consultați [Aflați despre Microsoft 365 Groups](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
