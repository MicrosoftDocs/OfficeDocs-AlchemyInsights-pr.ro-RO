# <a name="add-users-to-a-sharepoint-group"></a>Adăugarea utilizatorilor la un grup SharePoint

Un grup SharePoint este o colecție de utilizatori care au același set de permisiuni pentru site-uri și conținut. Mai degrabă decât să atribuiți permisiuni unei persoane odată, puteți utiliza grupuri pentru a atribui în mod convenabil același nivel de permisiune mai multor persoane în același timp.

- [Adăugarea utilizatorilor la un grup](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#add-users-to-a-group)
- [Adăugarea sau eliminarea membrilor din Microsoft 365 Groups utilizând Centrul de administrare](https://docs.microsoft.com/microsoft-365/admin/create-groups/add-or-remove-members-from-groups)
