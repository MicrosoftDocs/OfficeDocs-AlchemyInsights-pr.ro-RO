# <a name="site-is-not-available"></a>Site-ul nu este disponibil

În cazul în care site-ul SharePoint sau OneDrive nu este disponibil pentru mai mulţi utilizatori, poate fi o problemă temporară de serviciu. Verifica[Serviciul sănătate tabloul de bord](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) pentru a vedea dacă chiriaşul dumneavoastră este afectat.
