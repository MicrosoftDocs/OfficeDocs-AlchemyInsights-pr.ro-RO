# <a name="hub-site-permissions"></a>Permisiunile site-ului hub

Gestionarea permisiunilor unui site hub depinde de tipul de site de bază. Dacă site-ul este un site de echipă conectat la grup, ar trebui să gestionați permisiunile prin intermediul grupului Microsoft 365. Dacă este un site de comunicare, trebuie să gestionați permisiunile prin grupurile SharePoint.

- [Permisiunile site-ului hub](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- [Planificarea site-urilor hub](https://docs.microsoft.com/sharepoint/planning-hub-sites)
