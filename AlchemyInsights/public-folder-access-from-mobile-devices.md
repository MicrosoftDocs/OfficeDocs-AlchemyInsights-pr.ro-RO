# <a name="public-folder-access-from-mobile-devices"></a>Acces foldere publice de pe dispozitive mobile

Accesul la folderele publice nu funcționează cu ajutorul dispozitivelor mobile, utilizați OWA (Outlook pe web) sau Outlook pentru Windows desktop sau Outlook pentru Mac pentru a accesa folderele publice.

Dacă doriți să aveți funcționalități similare cu folderele publice și să aveți conținut accesibil pe dispozitivele mobile, consultați [Aflați despre grupurile Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) pentru o alternativă mai bună.
