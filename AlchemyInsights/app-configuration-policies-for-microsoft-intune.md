# <a name="app-configuration-policies-for-microsoft-intune"></a>Politici de configurare a aplicațiilor pentru Microsoft Intune

Politicile de configurare a aplicațiilor din Microsoft Intune oferă setări de configurare pentru aplicațiile iOS sau Android. Aceste setări de configurare permit o aplicație să fie particularizată. Suportul pentru politica de configurare este limitat de versiunile și aplicațiile sistemului de operare.

Pentru mai multe detalii, consultați [Politicile de configurare a aplicațiilor pentru Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
