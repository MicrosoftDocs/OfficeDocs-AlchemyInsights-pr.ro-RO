# <a name="enable-seamless-sso"></a>Activare SSO fără sudură

Azure Active Directory (Azure AD) fără sudură singur Sign-On (Fără sudură SSO) automat semne în utilizatori atunci când acestea sunt pe desktop-urile lor corporative care sunt conectate la rețeaua companiei.
  
[Urmați pașii din acest articol pentru a activa SSO fără sudură](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
