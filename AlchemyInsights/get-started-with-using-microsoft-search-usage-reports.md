# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Începeți să utilizați rapoartele de utilizare Microsoft Search

Rapoartele de utilizare a căutării Microsoft oferă detalii despre modul în care funcționează căutarea în organizația dvs. Aceste detalii vă vor ajuta să găsiți conținut și să căutați într-o experiență mai utilă și mai plăcută pentru utilizatorii din organizațiile dvs.

Pentru mai multe informații, consultați [Rapoartele de utilizare a căutării Microsoft.](https://go.microsoft.com/fwlink/?linkid=2152048)
