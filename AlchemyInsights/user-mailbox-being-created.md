# <a name="your-users-mailbox-is-being-created"></a>Cutia poștală a utilizatorului este creată

Poate dura până la o oră înainte ca o nouă cutie poștală să fie gata de utilizare.
  
Dacă cutia poștală a utilizatorului a fost "pregătirea" pentru mai mult de o oră, puteți încerca ștergerea și re-crearea utilizatorului, sau utilizați opțiunea "Să te sun" pentru a vorbi cu un inginer de asistență.
