# <a name="pdf-reader-app-in-microsoft-edge"></a>Aplicația PDF Reader în Microsoft Edge

Popularitatea fișierelor PDF evidențiază necesitatea unui cititor PDF fiabil, sigur și puternic, care poate fi adoptat de companiile Enterprise. Cititorul de PDF-caracteristici bogat încorporat în Microsoft Edge (versiunile 77 și versiunile mai recente) este doar o astfel de aplicație. Acesta funcționează în Windows și macOS și poate fi utilizat pentru a deschide și a adnota fișiere PDF care sunt locale, online sau încorporate în pagini web.

Pentru mai multe informații, consultați [PDF Reader în Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-pdf).
