# <a name="how-to-select-all-in-outlookcom"></a>de a selecta toate în Outlook.com

Aveți posibilitatea să selectați toate numai dacă toate mesajele sunt vizibile. Aceasta este pentru a preveni acțiunile accidentale asupra mesajelor pe care nu le puteți vedea.

**CTRL + faceți clic pe** selectează mai multe mesaje individuale.

**SHIFT + faceți clic pe** selectează toate mesajele dintre primul și ultimul mesaj pe care îl selectați.
