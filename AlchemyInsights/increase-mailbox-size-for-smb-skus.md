# <a name="upgrade-plans-to-increase-mailbox-size"></a>Planuri de upgrade pentru a mări dimensiunea cutiei poștale

Dacă dimensiunea cutiei poștale este 50 GB, puteți mări dimensiunea cutiei poștale la 100 GB comutând la [office 365 Enterprise E3](https://products.office.com/business/office-365-enterprise-e3-business-software) sau [Office 365 Enterprise E5](https://products.office.com/business/office-365-enterprise-e5-business-software). Iată pașii pentru a comuta la un alt abonament:
  
1. În centrul de administrare Microsoft 365, accesați pagina **Facturare**  >  [produse](https://go.microsoft.com/fwlink/p/?linkid=842054) .

2. Selectați abonamentul pentru upgrade, apoi selectați fila **upgrade** pentru a vedea lista de planuri noi disponibile.

3. Alegeți un plan pentru a vedea costul total nou pe lună. Asigurați-vă că citiți **informațiile esențiale**, apoi alegeți **Următorul** pentru a verifica.

4. Atunci când terminați de făcut o verificare, Microsoft 365 durează câteva minute pentru a finaliza comutatorul. Puteți începe să utilizați noul abonament imediat.

Pentru mai multe detalii, consultați [upgrade-ul la un alt plan](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/upgrade-to-different-plan).
