# <a name="desktop-app-for-sharepoint"></a>Aplicația desktop pentru SharePoint

Organizația este introdusă în programul de examinare privată pentru aplicația desktop pentru SharePoint.

Microsoft nu acceptă momentan aplicația desktop pentru SharePoint (examinare privată). Utilizați partenerii Microsoft private Preview înrolați pentru a obține asistență cu aplicația desktop pentru SharePoint.
