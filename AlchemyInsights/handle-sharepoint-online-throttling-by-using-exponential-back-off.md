# <a name="handle-sharepoint-online-throttling-by-using-exponential-back-off"></a>Tratarea limitării SharePoint Online prin utilizarea exponențială a retragerii

Atunci când vă amânați în SharePoint Online, tehnica de back-off exponențială așteaptă perioade de timp mai lungi progresiv înainte de a reîncerca codul care a fost modificat.
