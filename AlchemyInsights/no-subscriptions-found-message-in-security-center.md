# <a name="no-subscriptions-found-message-in-the-security-center"></a>Nu a fost găsit niciun abonament în centrul de securitate

Dacă în timp ce accesați centrul de securitate Microsoft Defender, primiți un mesaj "nu s-a găsit niciun abonament", înseamnă că Azure Active Directory (AAD) utilizat pentru a vă conecta utilizatorul la portal nu are o licență ATP Microsoft Defender.  

Licențele Windows E5 și Office E5 sunt licențe separate.

Deschideți un caz de asistență dacă licența a fost achiziționată, dar nu a fost furnizată în această instanță AAD. Fie aveți: <br/>
-   O posibilă problemă de furnizare A licențelor.<br/>
-   Ați furnizat din greșeală licența către un alt Microsoft AAD decât cel utilizat pentru autentificarea în serviciu.
