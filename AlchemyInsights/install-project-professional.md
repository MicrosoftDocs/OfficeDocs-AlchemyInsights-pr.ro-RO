# <a name="install-project-professional"></a>Instalați Project Professional

A se vedea [Proiectul instala](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281?ui=en-US&rs=en-US&ad=US) pentru a învăţa cum să instalaţi versiunea de abonament de proiect Online, şi versiunea fără abonament a proiectului 2019, 2016, sau 2013. 

Pentru Project 2010 sau 2007, urmaţi paşii din [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) sau [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Proiectul este vândut ca o aplicaţie de sine stătătoare, Deci, dacă nu aveţi încă, [cumpara sau încercaţi proiectului](https://products.office.com/project).
