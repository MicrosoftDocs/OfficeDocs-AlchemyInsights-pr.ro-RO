# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Găsirea rezultatelor în căutarea Bing utilizând aplicații logice Azure

Utilizând căutarea Bing din interiorul unei aplicații logice, puteți să automatizați procesarea rezultatelor căutării și să faceți aceste rezultate disponibile pentru alte acțiuni. De exemplu, puteți găsi elemente de știri, apoi Twitter le va publica ca tweeturi în fluxul dvs.

Pentru a afla mai multe, consultați [Găsirea rezultatelor în căutarea Bing utilizând aplicații logice Azure](https://go.microsoft.com/fwlink/?linkid=2151928).
