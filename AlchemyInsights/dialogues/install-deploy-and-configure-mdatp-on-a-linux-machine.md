# <a name="install-deploy-and-configure-mdatp-on-a-linux-machine"></a>Instalarea, implementarea și configurarea MDATP pe o mașină Linux

1. Asigurați-vă că sunteți abonat la MDATP și aveți acces la [portalul MDATP](https://go.microsoft.com/fwlink/?linkid=2144512).
2. Implementați MDATP [manual](https://go.microsoft.com/fwlink/?linkid=2144809) sau utilizând un instrument de gestionare a configurațiilor: [marionetă](https://go.microsoft.com/fwlink/?linkid=2144715) sau [Ansible](https://go.microsoft.com/fwlink/?linkid=2144716).

Pentru a afla mai multe, consultați [cum se instalează MDATP pentru Linux](https://go.microsoft.com/fwlink/?linkid=2144717).
