# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Bordul unui dispozitiv Windows 10 la protecția avansată a amenințărilor la Microsoft Defender (MDATP)

MDATP acceptă următoarele instrumente și metode de implementare pentru Windows 10:

- Politica de grup
- Microsoft Endpoint Configuration Manager
- Gestionarea dispozitivelor mobile, inclusiv Microsoft Intune
- Scripturi locale

Pentru a afla mai multe, consultați [Instrumente de bord și metode pentru dispozitivele Windows 10](https://go.microsoft.com/fwlink/?linkid=2143460).
