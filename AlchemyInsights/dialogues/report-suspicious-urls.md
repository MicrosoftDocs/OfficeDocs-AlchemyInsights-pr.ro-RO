# <a name="report-suspicious-urls"></a>Raportarea URL-urilor suspecte

Pentru a trimite URL-uri suspecte, urmați acești pași:

1. În flyout **Remitere nouă** , introduceți adresa URL completă, inclusiv protocolul (https://).
2. Dacă selectați **ar fi trebuit să fie blocat**, Specificați dacă adresa URL este **phishing** sau **malware**.
3. Selectați **Remitere**.
