# <a name="how-to-type-special-characters-on-a-mac"></a>Cum să tip caractere speciale pe un Mac

Pentru a tasta caractere speciale pe Mac, deschideţi meniul de **caractere speciale** cu claviatură şort ⌘ **comanda** + **Control** + **spatiu**.

O scurtătură rapid pentru @ este, de asemenea, **ALT** + **64**.
