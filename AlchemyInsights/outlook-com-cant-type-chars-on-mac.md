# <a name="how-to-type-special-characters-on-a-mac"></a>Cum se tastează caracterele speciale pe un Mac

Pentru a tasta caractere speciale pe Mac, deschideți meniul **caractere speciale** cu **comanda**comandă rapidă de la tastatură ⌘  +  **Control**  +  **Space**.

O comandă rapidă rapidă pentru @ este, de asemenea, **alt**  +  **64**.
