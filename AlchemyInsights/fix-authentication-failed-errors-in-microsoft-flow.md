# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Remedierea erorilor "Autentificare nereușită" în Microsoft Flow

Dacă primiți un mesaj de eroare "Autentificare nereușită" atunci când încercați să vă conectați la Microsoft Flow, este pentru că serviciul Flow este dezactivat în Azure Active Directory. Pentru detalii și pași pentru remedierea erorii, consultați [Nu vă puteți conecta la Microsoft Flow și primiți o eroare "Autentificare nereușită".](https://support.microsoft.com/help/4316891)
