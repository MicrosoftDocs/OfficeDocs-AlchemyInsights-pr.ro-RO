# <a name="create-a-site"></a>Crearea unui site

- Accesați [pagina site-uri din centrul de administrare Microsoft 365](https://portal.office.com/adminportal/home#/SitesList), apoi selectați **Adăugați un site**. 
    
- În noul centru de administrare SharePoint, pe pagina de site-uri, faceți clic pe **Creare**. 
    
 **Notă:** Imposibil de creat un site nou care are același URL cu un site existent. 
  
Pentru mai multe informații despre crearea unui site, consultați [crearea unui site](https://go.microsoft.com/fwlink/?linkid=866295).
  
Pentru informații despre permiterea utilizatorilor să creeze propriile site-uri, consultați [gestionarea creării site-ului în SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
