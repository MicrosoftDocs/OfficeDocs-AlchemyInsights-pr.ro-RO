# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Depanarea Outlook mobile setup pentru o cutie poştală partajată

Nu pot accesa cutia poştală partajată din Outlook app care este instalat pe dispozitivul mobil. Vrei această caracteristică? Lasa echipa de Outlook ingineri ştiu. [Voteaza aici](https://go.microsoft.com/fwlink/?linked=862116)!
  
Puteţi utiliza numai o cutie poştală partajată pe un dispozitiv mobil de deschide browser-ul, sign in la Office 365, şi apoi merge Outlook de pe web. Pentru instrucţiuni, consultaţi [adăugarea unei cutii poştale partajate în Outlook pe web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). Din Outlook pe web veţi putea accesa cutia poştală partajată.
  
Afla mai multe: [obţine ajutor în app pentru Outlook pentru iOS şi Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
