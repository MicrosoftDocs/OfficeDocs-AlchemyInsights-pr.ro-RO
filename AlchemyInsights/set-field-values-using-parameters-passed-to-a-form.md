# <a name="set-field-values-using-parameters-passed-to-a-form---lookup-fields"></a>Setarea valorilor câmpurilor utilizând parametrii trecuți într-un formular - câmpuri de căutare

Setați valoarea pentru câmpurile de căutare și Exemplu: Utilizați Xrm.Utility.openEntityForm pentru a deschide o fereastră nouă. Nu puteți seta valorile pentru lista de petreceri sau pentru căutarea.

Pentru mai multe informații, consultați [Setarea valorilor pentru câmpurile de căutare](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/developers-guide/gg334375(v=crm.8)#set-values-for-lookup-fields).
