# <a name="check-teams-service-status"></a>Verificare stare serviciu echipe

Pentru a verifica starea curentă a serviciului Microsoft Teams, accesați Tabloul de bord de sănătate al serviciului pentru centrul de administrare Microsoft 365.

Pentru mai multe informații, consultați [Cum se verifică starea serviciului Microsoft 365](https://docs.microsoft.com/office365/enterprise/view-service-health).
