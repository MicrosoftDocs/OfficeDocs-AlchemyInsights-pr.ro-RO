# <a name="software-updates-are-not-being-deployed"></a>Actualizările de software nu sunt implementate

Dacă ați configurat actualizări de software printr-un inel de actualizare Windows 10, dar actualizările nu sunt implementate, încercați următoarele:  

- Luați în considerare schimbarea serviciulului Windows de la un tip de lansare **semianual de canal** la un tip de lansare mai strict și mai frecvent.
- Verificați perioada de amânare pentru **actualizarea calității** și **actualizarea caracteristicilor**. O perioadă de amânare ar putea duce la o întârziere a actualizărilor de până la 180 de zile.
