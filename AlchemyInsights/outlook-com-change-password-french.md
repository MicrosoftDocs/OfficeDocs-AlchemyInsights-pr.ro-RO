# <a name="changer-votre-mot-de-passe-dans-outlookcom"></a>Schimbător votre mot de passe dans Outlook.com

Votre mot de passe Outlook.com est le puse que votre mot de passe Compte Microsoft.

1. Accédez à [https://account.microsoft.com/security](https://go.microsoft.com/fwlink/p/?linkid=842325&amp;clcid=0x409) et selectaţi **modificator le mot de treci**.
2. Par masura de sécurité, vous pouvez invité être à confirmer cu votre identité à l'aide d'un codul de sécurité.
3. Entrez votre mot de passe actuel, votre nouveau mot de passe, selectaţi puis **Enregistrer**.

Si vous clara_ oublié votre mot de passe, accédez à la [filme de réinitialisation de mot de treci de compte Microsoft](https://go.microsoft.com/fwlink/p/?linkid=841909), puis suivez les invită.

Si vous pensez que votre Outlook.com un été piraté, voir [Mon compte Outlook.com un été piraté](https://support.office.com/fr-fr/article/mon-compte-outlook-com-a-été-piraté-35993ac5-ac2f-494e-aacb-5232dda453d8).
