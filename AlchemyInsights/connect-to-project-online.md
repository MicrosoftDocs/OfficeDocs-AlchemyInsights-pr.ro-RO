# <a name="connect-to-project-online"></a>Conectarea la Project Online

Vă puteți conecta la Project Online utilizând:

- Project Online Desktop pentru a vă conecta la Project Online în mediul Office 365. Pentru mai multe informații, consultați [Conectarea la Project Online cu clientul Desktop Project Online](https://docs.microsoft.com/projectonline/connect-to-project-online-with-the-project-online-desktop-client).  
- Serviciul Power BI pentru a debloca informații din Project Web App pentru a ajuta la gestionarea proiectelor, portofoliilor și resurselor. Pentru mai multe informații, consultați [Conectarea la Project Web App cu Power BI](https://docs.microsoft.com/power-bi/service-connect-to-project-online).
