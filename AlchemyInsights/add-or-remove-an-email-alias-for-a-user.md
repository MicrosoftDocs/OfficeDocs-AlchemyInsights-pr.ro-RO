# <a name="add-or-remove-an-email-address-for-a-user"></a>Adăugarea sau eliminarea unei adrese de e-mail pentru un utilizator

[Urmați acești pași](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) în centrul de administrare Microsoft 365 pentru a începe.

 **Notă:** Puteți avea mai multe adrese de e-mail pentru un utilizator, dar aliasuri suplimentare *de e-mail* pot fi utilizate numai pentru primirea e-mailului. Dacă trebuie să trimiteți e-mailuri cu un alias, creați o [cutie poștală partajată](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox). În acest fel veți obține aliasul pentru a trimite și a primi e-mailuri.
