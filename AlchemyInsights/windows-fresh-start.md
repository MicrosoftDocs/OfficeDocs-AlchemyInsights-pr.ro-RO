# <a name="windows-fresh-start"></a>Pornire nouă windows

Acțiunea dispozitivului Windows Fresh Start oferă o alternativă la acțiunile Ștergere dispozitiv și Retragere dispozitiv, împreună cu opțiunea de a păstra starea de înscriere a dispozitivului împreună cu aplicațiile și setările folderului de pornire al utilizatorului.

Pentru mai multe informații, consultați [Utilizarea Unui nou început pentru a reseta dispozitivele Windows 10 cu Intune](https://docs.microsoft.com/intune/device-fresh-start).
