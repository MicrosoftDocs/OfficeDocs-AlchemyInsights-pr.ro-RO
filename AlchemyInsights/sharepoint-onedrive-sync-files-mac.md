# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>Instalarea, configurarea și sincronizarea fișierelor OneDrive sau SharePoint pe Mac 

Instalați, configurați și sincronizați OneDrive sau fișiere SharePoint pe un Mac cu instrucțiunile din [fișierele de sincronizare cu clientul de sincronizare OneDrive pe Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Dacă aplicația OneDrive este instalată și configurați pe un Mac, dar utilizatorii întâmpină probleme de sincronizare, consultați [remedierea problemelor de sincronizare cu OneDrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756).

> [!NOTE]
> Caracteristica mutare de foldere cunoscute în OneDrive Sync pentru Windows nu este disponibilă pentru Mac.
