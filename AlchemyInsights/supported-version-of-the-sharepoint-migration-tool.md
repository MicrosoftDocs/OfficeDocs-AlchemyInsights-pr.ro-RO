# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Versiune acceptată a instrumentului de migrare SharePoint



În prezent, instrumentul de migrare SharePoint acceptă doar migrările din SharePoint 2013. Versiunile 2010 și 2016 nu sunt acceptate în acest moment.
  
Pentru mai multe informații, consultați [prezentarea instrumentului de migrare SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
