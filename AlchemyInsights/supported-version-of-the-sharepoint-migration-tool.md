# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Sprijinit versiune a instrumentului de migraţia SharePoint



În prezent instrumentul de migraţia SharePoint acceptă numai migraţia de la SharePoint 2013. Versiunile 2010 şi 2016 nu sunt acceptate în acest moment.
  
Pentru informaţii suplimentare, consultaţi [introducerea SharePoint Migration Tool](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
