# <a name="change-the-domain-for-microsoft-365-group"></a>Modificarea domeniului pentru Microsoft 365 Group

Dacă doriți să modificați domeniul pentru un grup Microsoft 365 în timpul creării, urmați pașii din [Alegerea domeniului de utilizat la crearea grupurilor Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups).

După ce grupul Microsoft 365 este deja creat, în prezent nu există nici o modalitate de a modifica domeniul. Singura modalitate este de a crea un nou Grup Microsoft 365 și ștergeți unul vechi, dacă este necesar (adrese de e-mail suplimentare pot fi adăugate prin Centrul de [administrare Exchange](https://outlook.office365.com/ecp).
