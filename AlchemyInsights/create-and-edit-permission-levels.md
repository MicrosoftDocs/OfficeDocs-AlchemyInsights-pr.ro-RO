# <a name="working-with-sharepoint-permission-levels"></a>Lucrul cu nivelurile de permisiune SharePoint

SharePoint include niveluri implicite de permisiune pe care le puteți utiliza, cum ar fi **contribuția** și **vizualizarea numai**. Dar dacă una dintre acestea nu corespunde nevoilor dvs., puteți crea un nou nivel de permisiune. Rețineți că este mai ușor să țineți evidența nivelurilor de permisiune dacă nu modificați valorile implicite.

[Cum se creează și se editează nivelurile de permisiune](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
