# <a name="working-with-sharepoint-permission-levels"></a>Lucrul cu niveluri de permisiune SharePoint

SharePoint vine cu niveluri de permisiune implicite pe care le puteți utiliza, ar fi **contribuția** și **vizualizarea numai**. Dar dacă una dintre acestea nu satisface nevoile dvs., puteți crea un nou nivel de permisiune. Rețineți că este mai ușor să urmăriți nivelurile de permisiune dacă nu modificați valorile implicite.

[se creează și se editează nivelurile de permisiune](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
