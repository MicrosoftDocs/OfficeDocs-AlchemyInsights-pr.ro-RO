# <a name="working-with-sharepoint-permission-levels"></a>Lucrul cu nivelurile de permisiune SharePoint

SharePoint este livrat cu niveluri de permisiune implicite care se pot utiliza, ar fi **numai Contribuire** și **vizualizare**. Dar dacă una dintre acestea nu vă satisface nevoile, puteți crea un nou nivel de permisiune. Rețineți că este mai ușor să urmăriți nivelurile de permisiune dacă nu modificați valorile implicite.

[se creează și se editează nivelurile de permisiune](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
