# <a name="owner-cannot-create-sub-folder-using-outlook"></a>Proprietarul nu poate crea subfolder utilizând Outlook

**Există o problemă în curs de desfășurare cu proprietarii de foldere publice care creează subfoldere utilizând Outlook. Problema va fi rezolvată în curând.**

Între timp, utilizați una dintre următoarele soluții:

1. Utilizați Outlook pentru MAC pentru a crea subfolderul, deoarece problema afectează numai Outlook pentru ferestrele desktop (toate versiunile)
2. Au admin crea subfolder folosind EXO Shell sau EAC
3. Modificarea defaultPublicFolderMailbox/EffectivePublicFolderMailbox pe utilizator la altă cutie poștală decât cutia poștală de conținut pentru folderul care cauzează problema  
    - *Setare-cutie poștală utilizator1 implicitPublicFolderMailbox PubMBX3*
4. Așteptați o oră, reporniți clientul Outlook
