# <a name="live-events-in-microsoft-stream"></a>Evenimente în direct în Microsoft Stream

Puteți crea evenimente în direct utilizând Microsoft Stream în cadrul organizației. Puteți planifica, produce și livra evenimente în direct pentru o varietate de scenarii, cum ar fi evenimente la nivel de firmă, actualizări de conducere și altele.

- [Crearea unui eveniment în direct în Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Începeți lucrul cu evenimentele live din Microsoft teams](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Configurarea codificării pentru streaming în timp real în Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
