# <a name="live-events-in-microsoft-stream"></a>Evenimente live în Microsoft Stream

Aveți posibilitatea să creați evenimente live utilizând Microsoft Stream în întreaga organizație. Puteți programa, produce și livra evenimente live pentru o varietate de scenarii, ar fi evenimente la nivel de companie, actualizări de conducere și multe altele.

- [Evenimente live în Microsoft Stream](https://docs.microsoft.com/stream/live-event-overview)
- [Crearea unui eveniment live în Microsoft Stream](https://docs.microsoft.com/stream/live-create-event)
- [Introducere în evenimente live Microsoft Teams](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Configurarea codificatoarelor pentru streaming live în Microsoft Stream](https://docs.microsoft.com/stream/live-encoder-setup)
