# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Hartă o bibliotecă SharePoint la o unitate de reţea

Maparea unei biblioteci ca o unitate de reţea este temporară şi susţinute doar prin Internet Explorer. În schimb, [sincroniza SharePoint fişiere cu noul client de sincronizare OneDrive](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) care provides [Dosar On-Demand](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Acces la toate fişierele în OneDrive fără a utiliza spaţiu de stocare locale.
  
[Obţine ajutor Configurarea şi depanarea unităţi de reţea mapate](https://go.microsoft.com/fwlink/?linkid=872946)
