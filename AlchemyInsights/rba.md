# <a name="about-room-mailboxes"></a>Despre cutiile poștale ale camerei

O cutie poștală de cameră este o cutie poștală de resurse care este atribuită unei locații fizice, ar fi o sală de conferințe, o sală de clasă sau o sală de instruire. Pentru mai multe informații despre cutiile poștale de cameră, consultați [Crearea și gestionarea cutiilor poștale de cameră](https://go.microsoft.com/fwlink/p/?linkid=717533).
