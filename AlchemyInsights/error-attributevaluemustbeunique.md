1. <span data-ttu-id="c32ad-p102">Identifica proxyAddresses duplicat, userPrincipalName sau altă valoare de atribut care provoacă eroarea. 2. <span data-ttu-id="c32ad-108">Identificarea obiectului care ar trebui să continue să aibă valoarea duplicat şi obiect care ar trebui să nu.</span><span class="sxs-lookup"><span data-stu-id="c32ad-108">Identify which object should continue to have the duplicated value and which object should not.</span></span>

4. <span data-ttu-id="c32ad-112">În cazul în care aţi făcut schimbarea în incinta pe AD, să Azure AD conecta sincronizare schimbarea de eroare pentru a obţine stabilite.</span><span class="sxs-lookup"><span data-stu-id="c32ad-112">If you made the change in the on premises AD, let Azure AD Connect sync the change for the error to get fixed.</span></span>
