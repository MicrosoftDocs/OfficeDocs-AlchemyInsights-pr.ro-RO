# <a name="remove-an-extension-from-microsoft-edge"></a>Eliminarea unei extensii din Microsoft Edge

Pentru a elimina o extensie din Microsoft Edge:

1. Deschideți Microsoft Edge.
2. Selectați **extensiile de >...**
3. Pe pagina **extensii instalate** , găsiți extensia pe care doriți să o eliminați, apoi selectați **Eliminare** de lângă ea.
4. În caseta de dialog de confirmare, selectați **Eliminare**.
