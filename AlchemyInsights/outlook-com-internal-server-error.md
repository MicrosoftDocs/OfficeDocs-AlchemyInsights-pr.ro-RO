# <a name="internal-server-errors-in-outlookcom"></a>Erorile de server interne din Outlook.com

Oh, nu! Dacă vedeți mesajul "eroare 500" sau "eroare internă de server", acesta este al nostru și nu dumneavoastră-ne pare rău.

Consultați [pagina stare serviciu Office](https://portal.office.com/servicestatus) pentru a vă asigura că nu se întâmplă nimic altceva, dar cea mai bună soluție este să luați o pauză câteva minute, apoi să reîmprospătați browserul apăsând **Ctrl** și **F5**.
