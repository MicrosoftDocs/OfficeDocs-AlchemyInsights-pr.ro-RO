# <a name="edit-a-read-only-file"></a>Editarea unui fișier doar în citire

În bibliotecile de documente, când faceți clic pe numele unui document, este posibil ca programul Office să îl deschidă în mod neașteptat ca fișier doar în citire. Pentru a o deschide pentru editare, faceți clic pe **Deschidere** (cele trei puncte), apoi faceți clic pe **Editare.**
