# <a name="edit-a-read-only-file"></a>Edita un fişier doar în citire

În bibliotecile de documente, când faceţi clic pe numele unui document, programul Office ar putea în mod neaşteptat deschideţi-l ca fişier doar în citire. Pentru a deschide-l pentru editare, faceţi clic pe **deschide** (trei puncte), şi apoi faceţi clic pe **editez**
