# <a name="edit-a-read-only-file"></a>Editarea unui fișier doar în citire

În bibliotecile de documente, când faceți clic pe numele unui document, programul Office îl poate deschide în mod neașteptat ca fișier doar în citire. Pentru a o deschide pentru editare, faceți clic pe **Deschidere** (cele trei puncte), apoi faceți clic pe **Editare.**
