# <a name="recover-reinstall-or-restore-windows"></a>Recuperarea, reinstalarea sau restaurarea Windows

Pentru mai multe informații: 

- Pentru toate opțiunile de recuperare și soluțiile la problemele de recuperare comune, consultați [Opțiuni de recuperare în Windows 10](https://support.microsoft.com/windows/recovery-options-in-windows-10-31ce2444-7de3-818c-d626-e3b5a3024da5#bkmk_section7).

- Pentru a găsi cheia de recuperare BitLocker, consultați [Găsirea cheii de recuperare BitLocker](https://support.microsoft.com/windows/find-my-bitlocker-recovery-key-fd2b3501-a4b9-61e9-f5e6-2a545ad77b3e).

- Pentru a recupera fișierele pierdute dintr-o copie de rezervă, consultați [recuperarea fișierelor pierdute în Windows 10](https://support.microsoft.com/windows/recover-lost-files-on-windows-10-61f5b28a-f5b8-3cc2-0f8e-a63cb4e1d4c4).

- Pentru a face backup PC-ului la o unitate externă sau la o locație de rețea, consultați [copierea de rezervă și restaurarea în Windows 10](https://support.microsoft.com/windows/backup-and-restore-in-windows-10-352091d2-bb9d-3ea3-ed18-52ef2b88cbef).

- Pentru opțiunile de recuperare Windows 8,1, consultați [cum să reîmprospătați, să resetați sau să restaurați PC](https://support.microsoft.com/windows/how-to-refresh-reset-or-restore-your-pc-51391d9a-eb0a-84a7-69e4-c2c1fbceb8dd)-ul.
