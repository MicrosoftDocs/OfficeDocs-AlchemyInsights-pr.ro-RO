# <a name="onedrive-site-provisioning"></a>Provisioning site-ul OneDrive

Pentru conturile nou create, utilizatorii trebuie să se autentifică la centrul de administrare Microsoft 365 cu propriul cont și faceți clic pe pictograma OneDrive pentru a porni manual furnizarea site-ului OneDrive.

Cu toate acestea, dacă utilizatorul a făcut deja clic pe pictograma OneDrive și site-ul său personal nu este furnizat în termen de 24 de ore, vă rugăm să conectați un bilet de suport pentru asistență suplimentară.
