# <a name="identify-your-tenant-id"></a>Identificați ID-ul entității găzduite

Găsiți ID-ul organizației Microsoft 365 (ID-ul entității găzduite) în Centrul de administrare Azure.

1. Din pagina [Proprietăți,](https://aka.ms/AzurePropertiesPage)accesați caseta **ID** Director.

2. Copiați ID-ul după cum este necesar.

Pentru mai multe informații, consultați [Găsirea ID-ului organizației Microsoft 365.](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id)
