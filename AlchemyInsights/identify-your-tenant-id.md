# <a name="identify-your-tenant-id"></a>Identificarea ID-ului de entitate găzduită

Găsiți ID-ul organizației Microsoft 365 (ID-ul entității găzduite) în centrul de administrare Azure.

1. Din [pagina de proprietăți](https://aka.ms/AzurePropertiesPage), accesați caseta **ID Director** .

2. Copiați ID-ul după cum este necesar.

Pentru mai multe informații, consultați [Găsirea ID-ului de organizație Microsoft 365](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id).
