# <a name="creating-inbox-rules-for-shared-mailboxes"></a>Crearea de reguli Inbox pentru cutiile poştale partajate

Puteţi adăuga regulile pe o cutie poştală partajată aveţi acces într-un mod similar cu cât adăugați reguli pentru cont propriu.
  
Asigurați-vă că sunteţi un membru din cutia poştală partajată în cauză, şi apoi urmaţi paşii din acest articol: [Adauga reguli pentru o cutie poştală partajată](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164)
