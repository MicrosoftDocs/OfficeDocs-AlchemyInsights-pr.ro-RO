# <a name="what-is-autosave"></a>Ce este salvarea automată?

Salvare automată este o caracteristică disponibilă în Excel, Word și PowerPoint pentru abonații Microsoft 365 care vă salvează fișierul automat, la fiecare câteva secunde, pe măsură ce lucrați. 

Pentru întrebări frecvente, consultați [ce este salvarea automată?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
