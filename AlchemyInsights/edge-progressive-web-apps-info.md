# <a name="learn-about-progressive-web-apps-pwas-on-microsoft-edge-chromium"></a>Aflați despre aplicațiile web progresive (PWAs) din Microsoft Edge (crom)

PWAs sunt site-uri web progresiv îmbunătățite pentru a funcționa ca aplicațiile native pe platforme care le acceptă. PWAs combină cele mai bune calități ale aplicațiilor Web și native.

Din perspectiva unei platforme web, PWAs din [Microsoft Edge (crom)](https://go.microsoft.com/fwlink/?linkid=2135193) se bazează pe standarde. Aceștia permit utilizatorilor să instaleze aplicații direct din browser și nu necesită implementarea sau înregistrarea pe baza Microsoft Store. PWAsle desktop sunt acceptate pe orice platformă în care este disponibil Microsoft Edge (crom).
