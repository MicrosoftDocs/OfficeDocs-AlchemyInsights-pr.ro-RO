# <a name="unable-to-print"></a>Nu se poate imprima

Suntem conștienți de o problemă în care unii utilizatori nu pot imprima atunci când utilizează browsere Safari sau Edge. În timp ce lucrăm la o remediere pentru aceasta, vă rugăm să utilizați un alt browser. Ne cerem scuze pentru neplăcere.
