# <a name="powerpoint-slides-not-showing-in-skype-for-business-on-android-devices"></a>Diapozitivele PowerPoint nu se afișează în Skype for Business pe dispozitive Android

Dacă întâmpinați o problemă cu diapozitivele PowerPoint care nu se afișează pe dispozitiveAndroid, acest lucru se poate datora unei erori în browserul Chrome.

Încearcă să dezactivezi, apoi să dezinstalezi Chrome. După ce Chrome este dezinstalat, reactivați Chrome pentru a obține versiunea din fabrică (74.0.3729.136) în care problema este rezolvată.
