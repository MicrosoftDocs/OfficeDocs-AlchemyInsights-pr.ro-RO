# <a name="add-or-remove-a-custom-role"></a>Adăugarea sau eliminarea unui rol particularizat

Pentru a gestiona resursele Azure, consultați [adăugarea sau eliminarea atribuirilor de roluri Azure utilizând portalul Azure](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal).

Pentru a gestiona rolurile particularizate pentru Azure Active Directory (Azure AD), consultați [roluri de administrator particularizate în Azure Active Directory (Preview)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview).
