# <a name="prevent-microsoft-search-in-bing-from-installing-with-microsoft-365-apps"></a>Împiedicați Microsoft Search în Bing să instaleze cu aplicațiile Microsoft 365

Pentru a împiedica serviciul de fundal pentru Microsoft Search în Bing să instaleze cu Microsoft 365 Apps pentru întreprinderi, utilizați una dintre următoarele opțiuni:

- Instrumentul de implementare Office sau Politica de grup
- Microsoft Endpoint Configuration Manager (ramura curentă) sau Microsoft Intune

Pentru a afla mai multe, consultați [cum se împiedică instalarea serviciului de fundal pentru Microsoft Search în Bing cu aplicațiile microsoft 365 pentru întreprinderi](https://go.microsoft.com/fwlink/?linkid=2151946).
