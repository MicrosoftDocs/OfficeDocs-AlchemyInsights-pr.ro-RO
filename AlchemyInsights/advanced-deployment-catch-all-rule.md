# <a name="deploy-microsoft-365-apps"></a>Implementarea aplicațiilor Microsoft 365

Pentru implementarea aplicațiilor Microsoft 365 către mai mulți utilizatori finali prin intermediul instrumentului de implementare Office, Microsoft Endpoint Configuration Manager, ca parte a unei imagini de disc a sistemului de operare sau într-un scenariu de computer partajat utilizând Windows Server cu Remote Desktop Services (RDS) sau Terminal Services, luați în considerare utilizarea ghidului pas cu pas în **Microsoft 365 Apps for enterprise deployment advisor**.
  
Pentru a începe, asigurați-vă că v-ați conectat ca administrator de entitate găzduită sau administrator global, apoi faceți clic aici: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
