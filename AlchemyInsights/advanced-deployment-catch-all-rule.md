# <a name="deploy-office-apps"></a>Implementaţi Office apps

Pentru implementarea Office apps la mai multe utilizatorii finali prin intermediul instrument de implementare Office, centrul System Configuration Manager, ca parte a unei imagini de disc sistem de operare, sau într-un scenariu de computer partajat folosind Windows Server cu Remote Desktop servicii (RDS) sau Terminal Servicii, luaţi în considerare utilizarea ghid pas cu pas în **Office 365 ProPlus consultantul de implementare**.
  
Pentru a începe, asiguraţi-vă că sunteţi conectat ca un chiriaş admin sau administrator global, şi apoi faceţi clic aici: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
