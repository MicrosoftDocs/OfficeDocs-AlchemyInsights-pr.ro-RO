# <a name="deploy-microsoft-365-apps"></a>Implementarea aplicațiilor Microsoft 365

Pentru implementarea aplicațiilor Microsoft 365 la mai mulți utilizatori finali prin intermediul instrumentului de implementare Office, Microsoft Endpoint Configuration Manager, ca parte a unei imagini de disc a sistemului de operare sau într-un scenariu de computer partajat, utilizând Windows Server cu servicii Desktop la distanță (RDS) sau Terminal Services, luați în considerare utilizarea ghidului pas cu pas în **Microsoft 365 Apps for Enterprise Deployment Advisor**.
  
Pentru a începe, asigurați-vă că sunteți conectat ca administrator al entității găzduite sau ca administrator global, apoi faceți clic aici: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
