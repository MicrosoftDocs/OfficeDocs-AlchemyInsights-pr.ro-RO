# <a name="set-microsoft-edge-as-the-default-browser-on-a-domain-joined-device"></a>Setați Microsoft Edge ca browser implicit pe un dispozitiv cu domeniu unit

Setați Microsoft Edge ca browser implicit: 

1. [Creați un fișier de configurare cu asocieri implicite și](https://go.microsoft.com/fwlink/?linkid=2132437) stocați-l local sau într-o partajare de rețea.

1. Deschideți editorul de Politică de grup, apoi accesați **Computer Configuration**  >  **Administrative Templates** Windows  >  **Components** File  >  **Explorer**.

1. Selectați **Setați un fișier de configurare cu asocieri implicite.**

1. Selectați **Setare** politică , apoi **selectați Activat**.

1. Sub **Opțiuni,** introduceți locația fișierului de configurare cu asocieri implicite, apoi selectați **OK.**
