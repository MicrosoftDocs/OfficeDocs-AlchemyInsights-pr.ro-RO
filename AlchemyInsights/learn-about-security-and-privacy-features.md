# <a name="learn-about-the-security-and-privacy-features-of-microsoft-search-in-bing"></a>Aflați despre caracteristicile de securitate și confidențialitate ale Microsoft Search în Bing

Caracteristicile de securitate și confidențialitate ale Microsoft Search în Bing ajutați o organizație să protejeze datele de utilizator și datele de la locul de muncă. De exemplu, solicitările sunt făcute prin HTTPS, iar conexiunile sunt criptate de la sfârșit la sfârșit.

Pentru mai multe informații, consultați [Securitate și confidențialitate pentru Microsoft Search în Bing](https://docs.microsoft.com/microsoftsearch/security-for-search).
