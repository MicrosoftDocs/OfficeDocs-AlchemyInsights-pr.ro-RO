# <a name="check-in-several-files-at-once"></a>Check-in mai multe fişiere simultan

Verificarea fişierelor în şi din SharePoint o bibliotecă este o modalitate foarte bună de a urmări şi de control cum conţinut este actualizat. Dacă doriţi să modificaţi un fişier de pe un site şi doriţi să vă asiguraţi că nimeni altcineva poate editaţi-l, extrageți fișierul. Atunci când aveţi fişierul verificat, puteţi edita este online sau offline, şi salvaţi-l pe mai multe ori, dacă este necesar.
