# <a name="bulk-delete-user"></a>Ștergere în bloc utilizator

Aveți posibilitatea să eliminați un număr mare de utilizatori din grup utilizând un fișier cu valori separate prin virgulă (CSV) pentru a șterge în bloc utilizatorii. Pentru a șterge utilizatorii în bloc care utilizează portalul Azure AD, consultați [Ștergerea în bloc a utilizatorilor din Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
