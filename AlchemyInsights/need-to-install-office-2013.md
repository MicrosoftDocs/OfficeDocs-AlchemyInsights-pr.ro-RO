# <a name="office-2013-is-no-longer-supported-in-microsoft-365-subscriptions"></a>Office 2013 nu mai este acceptat în microsoft 365 abonamente

Din 28 februarie 2017, Office 2013 nu mai este acceptat în abonamentele Microsoft 365.
  
Pentru a asigura securitatea suitei Office și pentru a obține toate cele mai recente caracteristici, instalați gratuit cea mai recentă versiune de Office ca parte a abonamentului Microsoft 365. Consultați [Upgrade-ul utilizatorilor la cel mai recent client Office](https://docs.microsoft.com/microsoft-365/admin/setup/upgrade-users-to-latest-office-client).
