# <a name="onedrive-cant-add-your-folder-right-now"></a>OneDrive puteți adăuga folderul chiar acum

Eroarea "Ne pare rău, OneDrive nu poate adăuga folderul chiar acum" poate apărea dacă aveți biblioteci cu coloanele  Finalare sau Validare  sau când Securitate elemente schiță este setată fie la Numai utilizatorii care pot edita, fie la Numai utilizatorii care pot aproba elemente din biblioteca Setări.    

Pentru instrucțiuni privind eliminarea setărilor coloanei de blocare a sincronizării, consultați Restricții și [limite OneDrive și SharePoint](https://support.microsoft.com/office/64883a5d-228e-48f5-b3d2-eb39e07630fa).
