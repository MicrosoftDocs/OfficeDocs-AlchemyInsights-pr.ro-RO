# <a name="project-for-the-web-common-issues-and-resolutions"></a>Probleme uzuale în Project pentru web și rezolvări

Iată câteva subiecte solicitate frecvent despre Project pentru web:

- Aflați despre [Project pentru web](https://support.microsoft.com/ro-RO/office/what-is-project-for-the-web-c19b2421-3c9d-4037-97c6-f66b6e1d2eb5), disponibil în toate cele [trei abonamente Project](https://products.office.com/project/compare-microsoft-project-management-software).
- Activați sau dezactivați Project pentru web sau planul principal [pentru utilizatorii din organizația dvs.](https://docs.microsoft.com/project-for-the-web/turn-project-for-the-web-off). Administratorii pot controla accesul la Project pentru web prin pagina **Setări Project** din Centrul de administrare Microsoft.
- Aflați mai multe despre [ce puteți face](https://support.office.com/article/what-can-you-do-with-project-for-the-web-b30f5442-be5f-43d2-9072-c95bff778ea1) cu Project pentru web.
- Ajutați utilizatorii să-și creeze și să își construiască proiectele cu [Instruirea Project pentru web](https://support.office.com/article/get-started-with-project-for-the-web-50bf3e29-0f0d-4b7a-9d2c-7c78389b67ad).
- Familiarizați-vă cu [limitele](https://docs.microsoft.com/project-for-the-web/project-for-the-web-limits-and-boundaries) Project pentru web.

Pentru mai multe informații despre Project pentru web, consultați [ajutorul pentru administratori Project pentru web](https://docs.microsoft.com/project-for-the-web/projectforweb-admin-home).
