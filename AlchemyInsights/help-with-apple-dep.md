# <a name="help-with-apple-dep"></a>Ajutor cu Apple DEP

Dacă un dispozitiv DEP nu este capabil să se înregistreze și Mae (multi-factor Authentication) este activat, dezactivați Mae. Mae nu este acceptat momentan pentru înscriere DEP.
