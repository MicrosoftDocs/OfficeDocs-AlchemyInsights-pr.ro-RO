# <a name="turn-off-focused-inbox-in-email-clients"></a>Nap off concentrat Inbox poştă electronică clients

Pentru a dezactiva concentrat cutia poştală în Outlook 2016, Outlook Web, Outlook.com/Hotmail.com, Outlook 2016 pentru Mac, sau Mail pentru Windows 10, consultaţi [Dezactivarea concentrat Inbox](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx).
  
Pentru a dezactiva concentrat Inbox pentru toată lumea în organizaţie, a se vedea [Configurarea Inbox concentrat pentru toată lumea din organizaţia dvs](https://support.office.com/article/613a845c-4b71-41de-b331-acdcf5b6625d.aspx).
