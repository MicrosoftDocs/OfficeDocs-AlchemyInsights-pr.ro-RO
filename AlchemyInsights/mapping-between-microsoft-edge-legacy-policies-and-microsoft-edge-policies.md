# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Aflați despre maparea dintre politicile moștenite Microsoft Edge și politicile Microsoft Edge

Politicile de grup pentru Microsoft Edge Legacy (versiunile 45 și anterioare) și setările de gestionare a dispozitivelor mobile (MDM) se mapează la politicile relevante din Microsoft Edge, versiunea 80. Pentru mai multe informații, consultați [Microsoft Edge Legacy la maparea politicilor Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).
