# <a name="solutions-for-unsupported-operating-system"></a>Soluții pentru "sistem de operare neacceptat"

Atunci când încercați să instalați Microsoft 365, este posibil să vedeți un mesaj de eroare **neacceptat pentru sistemul de operare** dacă dispozitivul nu îndeplinește [cerințele de sistem](https://products.office.com/office-system-requirements).
  
Microsoft 365 necesită:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 sau Windows Server 2008 R2

    SAU

- Mac OS X 10,10

Dacă ați primit acest mesaj în timp ce încercați să instalați Office pe un dispozitiv mobil sau pe un Chromebook, consultați [cum să instalați și să ruleze Microsoft Office pe un Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
