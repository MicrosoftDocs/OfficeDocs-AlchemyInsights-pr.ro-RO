# <a name="proxyaddress-incorrect"></a>ProxyAddress incorecte

Atunci când un obiect este sincronizată cu Azur AD, valorile specificate în atributul proxyAddresses din Active Directory sunt comparate cu Azur AD reguli, şi apoi Atributul proxyAddresses este populată în Azur AD. Prin urmare, valorile din atributul proxyAddresses pentru obiectul din Active Directory nu poate fi la fel ca valorile din atributul proxyAddresses în Azur AD.
  
Pentru a afla mai multe despre cum proxyaddress este populat, a se vedea [cum atributul proxyAddress este populată în Azur AD](https://support.microsoft.com/en-us/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
