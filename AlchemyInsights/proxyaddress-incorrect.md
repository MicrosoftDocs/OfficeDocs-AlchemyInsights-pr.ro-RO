# <a name="proxyaddress-incorrect"></a>Adresă proxy incorectă

Atunci când un obiect este sincronizat cu Azure AD, valorile specificate în atributul proxyAddresses în Active Directory sunt comparate cu regulile Azure AD și apoi atributul proxyAddresses este populat în Azure AD. De aceea, valorile atributul proxyAddresses pentru obiectul în Active Directory nu poate fi la fel ca valorile atributului proxyAddresses în Azure AD.
  
Pentru a afla mai multe despre modul în care este populată adresa proxy, consultați [Cum este populat atributul proxyAddress în Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
