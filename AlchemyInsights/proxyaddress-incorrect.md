# <a name="proxyaddress-incorrect"></a>ProxyAddress incorect

Atunci când un obiect este sincronizat Azure AD, valorile care sunt specificate în atributul proxyAddresses în Active Directory sunt comparate cu regulile Azure AD și apoi Atributul proxyAddresses este populat în Azure AD. De aceea, valorile atributului proxyAddresses pentru obiectul în Active Directory nu poate fi la fel ca valorile atributului proxyAddresses în Azure AD.
  
Pentru a afla mai multe despre modul în care este populată proxyAddress, vedeți [atributul proxyAddress este populat în AZURE AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
