# <a name="why-cant-i-edit-this-file"></a>De ce nu pot edita acest fișier?

Dacă un document, o foaie de lucru sau o prezentare este blocat pentru editare și nu poate fi editat în aplicațiile desktop Windows, este posibil ca fișierul să fie editat de altă persoană sau să aibă fișierul extras.

Uitați-vă rapid la câteva dintre motivele pentru [care este posibil să nu reușiți să editați fișierul](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
