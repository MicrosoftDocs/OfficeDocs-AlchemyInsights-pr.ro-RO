# <a name="remove-users-from-a-sharepoint-group"></a>Eliminarea utilizatorilor dintr-un grup SharePoint

Un grup SharePoint este o colecție de utilizatori care au toate același set de permisiuni pentru site-uri și conținut. În loc să atribuiți permisiuni unei persoane simultan, aveți posibilitatea să utilizați grupuri pentru a atribui convenabil același nivel de permisiune multor persoane simultan.

- [Eliminarea utilizatorilor dintr-un grup](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#remove-users-from-a-group)
- [Adăugarea sau eliminarea membrilor din grupurile Office 365 utilizând centrul de administrare](https://docs.microsoft.com/office365/admin/create-groups/add-or-remove-members-from-groups?view=o365-worldwide)
