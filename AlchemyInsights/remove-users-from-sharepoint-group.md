# <a name="remove-users-from-a-sharepoint-group"></a>Eliminarea utilizatorilor dintr-un grup SharePoint

Un grup SharePoint este o colecție de utilizatori care au cu toții același set de permisiuni pentru site-uri și conținut. În loc să atribuiți permisiuni câte o persoană odată, puteți utiliza grupurile pentru a atribui în mod convenabil același nivel de permisiune mai multor persoane simultan.

- [Eliminarea utilizatorilor dintr-un grup](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions#remove-users-from-a-group)
- [Adăugarea sau eliminarea membrilor din grupurile Microsoft 365 utilizând centrul de administrare](https://docs.microsoft.com/microsoft-365/admin/create-groups/add-or-remove-members-from-groups)
