# <a name="outlook-search-not-indexing"></a>Căutarea Outlook nu se indexează

Suntem conștienți de problemele de căutare Outlook legate de actualizările de securitate recente pentru Windows.
  
Atunci când căutați în Outlook, este posibil să primiți una dintre aceste erori: **ceva nu a funcționat corect și căutarea nu a putut fi finalizată**sau **rezultatele căutării pot fi incomplete, deoarece elementele sunt încă indexate**. Remedierea a fost emisă ca actualizare Windows opțională. Vă rugăm să instalați una corespunzătoare versiunii de Windows pe care o vedeți în această problemă: 
  
> [Windows 7-27 iunie 2017 actualizare](https://support.microsoft.com/kb/4022168.aspx)
    
> [Actualizare Windows 8,1-27 iunie 2017](https://support.microsoft.com/kb/4022720.aspx)
    
> [Windows 10 versiunea 1703-27 iunie 2017 actualizare](https://support.microsoft.com/kb/4022716.aspx)
    
Pentru cele mai recente informații, consultați articolul: [probleme cunoscute în Outlook în problema actualizărilor de securitate din iunie 2017](https://support.office.com/article/Outlook-known-issues-in-the-June-2017-security-updates-3F6DBFFD-8505-492D-B19F-B3B89369ED9B.aspx) #5.
