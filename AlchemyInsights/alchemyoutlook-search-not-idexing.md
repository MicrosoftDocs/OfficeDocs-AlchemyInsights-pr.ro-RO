# <a name="outlook-search-not-indexing"></a>CăutareOutlook nu indexare

Suntem conștienți de problemele de căutare Outlook legate de actualizările de securitate recente pentru Windows.
  
Când căutați în Outlook, este posibil să obțineți una dintre aceste erori: **ceva nu a mers bine și căutarea nu a putut fi finalizată**sau Rezultatele **căutării pot fi incomplete, deoarece elementele sunt încă indexate**. Remedierea a fost emisă ca actualizare Windows opțională. Instalați-o pe cea corespunzătoare versiunii de Windows pe care o vedeți în: 
  
> [Actualizare Windows 7 - 27 iunie 2017](https://support.microsoft.com/kb/4022168.aspx)
    
> [Windows 8.1 - 27 iunie 2017 actualizare](https://support.microsoft.com/kb/4022720.aspx)
    
> [Windows 10 versiunea 1703 - 27 iunie 2017 actualizare](https://support.microsoft.com/kb/4022716.aspx)
    
Pentru cele mai recente informații, consultați articolul: [Probleme cunoscute Outlook în iunie 2017 actualizări de securitate](https://support.office.com/article/Outlook-known-issues-in-the-June-2017-security-updates-3F6DBFFD-8505-492D-B19F-B3B89369ED9B.aspx) Problemă #5.
