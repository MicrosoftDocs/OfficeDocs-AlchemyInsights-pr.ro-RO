# <a name="set-up-microsoft-information-protection"></a>Configurarea protecției informațiilor Microsoft

Obțineți o prezentare generală a capacităților pe care le puteți aplica strategiei de protecție a informațiilor, astfel încât să fiți siguri că informațiile sensibile sunt protejate. Utilizați o abordare a ciclului de viață în patru etape în care Descoperiți, clasificați, protejați și monitorizați informațiile sensibile. Ghidul "Configurarea protecției informațiilor Microsoft" oferă instrucțiuni pentru finalizarea fiecăruia dintre aceste etape.
