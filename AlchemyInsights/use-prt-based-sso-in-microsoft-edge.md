# <a name="use-prt-based-sso-in-microsoft-edge"></a>Utilizarea SSO bazat pe PRT în Microsoft Edge

Un [token de reîmprospătare primară (PRT)](https://go.microsoft.com/fwlink/?linkid=2133632) este o cheie Azure Active Directory utilizată pentru autentificare pe dispozitive Windows 10, iOS și Android. PRT activează SSO pentru toate aplicațiile utilizate pe aceste dispozitive.

În Microsoft Edge, suportul pentru SSO bazat pe PRT este nativ și nu necesită extensii suplimentare. În Windows 10 RS3 și mai sus, utilizatorii cu autentificare SSO vor primi SSO cu PRT la site-urile web care acceptă SSO bazat pe PRT.
