# <a name="create-a-new-folder-in-outlookcom"></a>Creați un folder nou în Outlook.com

1. Sub **foldere** din panoul de navigare, selectați **folder nou**.

2. Tastați un nume pentru folder și apăsați Enter.

Pentru a afla mai multe, consultați [crearea unui folder în Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
