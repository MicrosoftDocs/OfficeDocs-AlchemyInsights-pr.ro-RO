# <a name="create-a-new-folder-in-outlookcom"></a>Creaţi un folder nou în Outlook.com

1. La **foldere** din panoul de navigare, selectaţi **folder nou**.

2. Tastaţi un nume pentru dosar şi apăsaţi Enter.

Pentru mai multe informaţii, consultaţi [crearea unui folder in Outlook.com](https://support.office.com/article/5fa8de74-3562-4729-ac1d-5599f470b25a?wt.mc_id=Office_Outlook_com_Alchemy).
