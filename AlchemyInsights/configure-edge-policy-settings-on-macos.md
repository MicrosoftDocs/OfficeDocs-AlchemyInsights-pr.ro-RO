# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Utilizați bips pentru a configura setările de politică Microsoft Edge pe macOS

Pentru a configura Microsoft Edge (canalul stabil 81 sau o versiune mai recentă) pe macOS, utilizați un fișier vădit Microsoft Edge Policy pe bips Pro 10,19.

Pentru mai multe informații, consultați [Configurarea setărilor politicii Microsoft Edge pe MacOS cu bips](https://go.microsoft.com/fwlink/?linkid=2134761).
