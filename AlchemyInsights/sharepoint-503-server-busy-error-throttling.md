SharePoint Online foloseste supraîncărcarea pentru a menţine performanţe optime şi fiabilitate serviciului SharePoint Online. Supraîncărcarea limitele numărului de acţiunile utilizatorului sau concurente apeluri (de script sau cod) pentru a preveni suprasolicitarea resurselor. Throttling limits the number of user actions or concurrent calls (by script or code) to prevent overuse of resources.
