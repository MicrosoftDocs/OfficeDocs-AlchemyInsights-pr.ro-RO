# <a name="how-to-change-your-theme"></a>Cum se modifică tema

Pentru a modifica tema în Outlook.com sau Outlook pe web, accesați **Setări** și alegeți o temă sub **temă**. Pentru a vedea toate temele disponibile, selectați **Vedeți totul** sub **temă**.
