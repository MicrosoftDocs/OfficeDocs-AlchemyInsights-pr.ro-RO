# <a name="web-content-filtering"></a>Filtrarea conținutului web

Filtrarea conținutului web se află în prezent în previzualizare publică.

Această versiune preliminară este furnizată fără un acord la nivel de serviciu și nu este recomandată pentru volumul de producție. Este posibil ca anumite caracteristici să nu fie acceptate sau să aibă capacități constrânse. Pentru mai multe informații, consultați [Caracteristici de previzualizare Microsoft Defender pentru punct final.](/microsoft-365/security/defender-endpoint/preview)

Pentru a afla mai multe despre filtrarea conținutului web, consultați [Filtrarea conținutului web.](/microsoft-365/security/defender-endpoint/web-content-filtering)
