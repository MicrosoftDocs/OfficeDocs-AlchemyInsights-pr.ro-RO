# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Depanare scenarii refresh - placi de erori

Dacă vă confruntaţi cu ţiglă reîmprospătare erori în cadrul serviciului de putere BI, vă rog, aşteptaţi aproximativ 10-15 minute pentru refresh să fie reflectate în tigla de bord. În cazul în care acesta este încă nu manifestare sus, re-pin vizualizare la tabloul de bord.

Pentru mai multe informaţii:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
