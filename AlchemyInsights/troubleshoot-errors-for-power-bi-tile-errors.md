# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Depanare scenarii refresh - placi de erori

Vă rugăm să aşteptaţi aproximativ 10-15 minute pentru refresh să fie reflectate în tigla de bord. În cazul în care acesta este încă nu manifestare sus, re-pin vizualizare la tabloul de bord.
