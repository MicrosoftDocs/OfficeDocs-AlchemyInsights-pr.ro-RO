# <a name="set-up-azure-active-directory"></a>Configurarea Azure Active Directory

[Ghidul de configurare AZURE AD](https://go.microsoft.com/fwlink/?linkid=2134390) oferă informații pentru a vă asigura că organizația dumneavoastră are o fundație de securitate puternică. În acest ghid, veți configura caracteristicile inițiale, cum ar fi Azure RBAC Access Control (Azure RBAC) pentru administratori, Azure AD Connect pentru directorul local și Azure AD Connect Health, astfel încât să puteți monitoriza sănătatea identității hibride în timpul sincronizărilor automate.

De asemenea, include informații esențiale cu privire la activarea resetării parolelor cu autoservire, a accesului condiționat și a conectărilor integrate de la terți, inclusiv a protecției avansate a identității și a accesului la automatizarea utilizatorilor.
