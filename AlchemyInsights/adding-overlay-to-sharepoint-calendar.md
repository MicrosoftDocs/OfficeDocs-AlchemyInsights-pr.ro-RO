# <a name="adding-an-overlay-to-a-sharepoint-calendar"></a>Adăugarea unei suprapuneri la un calendar SharePoint

Calendarele SharePoint sunt tipuri de liste clasice care sunt uneori confundate cu calendarele de grup Office 365.
 
Dacă utilizați un calendar SharePoint pe site, aveți posibilitatea să suprapuneți propriul calendar Exchange pe acesta pentru a crea o vizualizare calendar combinată. Alternativ, aveți posibilitatea să suprapuneți un alt calendar SharePoint atâta timp cât ambele calendare SharePoint sunt în aceeași colecție de site-uri.
 
Pentru a adăuga o supralay la un calendar SharePoint clasic, urmați pașii din [Supralay un calendar SharePoint cu un calendar din Exchange sau SharePoint](https://support.office.com/article/Overlay-a-SharePoint-calendar-with-a-calendar-from-Exchange-or-SharePoint-4CAEBE59-3994-4A94-9322-B31ABB8A5E9A).
