# <a name="adding-an-overlay-to-a-sharepoint-calendar"></a>Adăugarea unei suprapuneri la un calendar SharePoint

Calendarele SharePoint sunt tipuri de liste clasice, care sunt uneori confundate cu Office 365 calendare de grup.
 
Dacă utilizați un calendar SharePoint pe site-ul dvs., aveți posibilitatea să suprapuneți propriul calendar Exchange pe el pentru a crea o vizualizare combinată calendar. Alternativ, puteți suprapune un alt calendar SharePoint atâta timp cât ambele calendare SharePoint sunt în aceeași colecție de site-ul.
 
Pentru a adăuga o suprapunere la un calendar SharePoint clasic, urmați pașii din [suprapunere un calendar SharePoint cu un calendar din Exchange sau SharePoint](https://support.office.com/article/Overlay-a-SharePoint-calendar-with-a-calendar-from-Exchange-or-SharePoint-4CAEBE59-3994-4A94-9322-B31ABB8A5E9A).
