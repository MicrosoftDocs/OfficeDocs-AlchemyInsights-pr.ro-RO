# <a name="yammer---possible-service-issue"></a>Problemă de serviciu posibilă pentru Yammer

Este posibil să existe o problemă temporară în care utilizatorii pot avea erori atunci când accesează Yammer. Verificați [tabloul de bord stare servicii](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) pentru a vedea dacă acest lucru afectează rețeaua.
