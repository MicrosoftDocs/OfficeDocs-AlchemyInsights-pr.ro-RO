# <a name="yammer---possible-service-issue"></a>Yammer - serviciu posibil problema

Poate fi o problemă temporară în cazul în care utilizatorii pot avea erori la accesarea Yammer. Verifica [Serviciul sănătate tabloul de bord](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) pentru a vedea dacă acest lucru afectează reţea.
