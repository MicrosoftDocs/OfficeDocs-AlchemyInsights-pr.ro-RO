# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>Nu puteți șterge fișiere sau fotografii din fișiere în Outlook pe web

**Fișiere** afișează fișierele și fotografiile atașate la mesajele din cutia poștală. Pentru a elimina fișiere și fotografii din fișiere, trebuie să ștergeți mesajele de e-mail la care sunt atașate. După ce ștergeți mesajele, trebuie, de asemenea, să le ștergeți din folderul Elemente șterse.
