# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>Nu se poate şterge fişiere sau fotografii la fişiere în Outlook pe web

**Fişiere** Arată fişierele şi fotografiile ataşate la mesajele din cutia poştală. Pentru a elimina fişiere şi fotografii de fişiere, aveţi nevoie pentru a şterge mesajele de e-mail, acestea sunt anexate. După ştergerea mesajelor, trebuie să ştergeţi-le din folderul Elemente şterse.
