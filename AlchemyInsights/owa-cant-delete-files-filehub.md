# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>Imposibil de șters fișiere sau fotografii din fișiere în Outlook pe web

**Fișierele** afișează fișierele și fotografiile atașate mesajelor din cutia poștală. Pentru a elimina fișiere și fotografii din fișiere, trebuie să ștergeți mesajele de e-mail la care sunt atașate. După ștergerea mesajelor, trebuie, de asemenea, să le ștergeți din folderul Elemente șterse.
