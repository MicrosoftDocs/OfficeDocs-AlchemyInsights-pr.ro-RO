# <a name="manage-synchronized-user-accounts"></a>Gestionați conturile de utilizator sincronizate

De obicei, utilizați Centrul de administrare Microsoft 365 pentru a adăuga și modifica conturile de utilizator. Cu toate acestea, nu se poate utiliza dacă directorul instantanee sau SSO sunt activate. În schimb, gestionați utilizatorii cu Windows PowerShell așa este descris în [configurați proprietățile contului de utilizator cu Office 365 PowerShell](https://docs.microsoft.com/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
