# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>Utilizatorii externi nu pot trimite e-mailuri la grupul Microsoft 365

Utilizați acești pași pentru a permite utilizatorilor externi să trimită prin e-mail grupul Microsoft 365:

1. Conectați-vă la [centrul](https://admin.microsoft.com/)de administrare  >  **Groups**  >  **Grupuri grupuri**.
2. Alegeți grupul pe care doriți să îl gestionați pentru a deschide panoul Setări.
3. Pe fila **Setări,** selectați **Se permite expeditorilor externi să trimită prin e-mail acest grup**.
4. Selectați **Salvare**.</br>
    Se lasă până la o oră pentru ca setările să fie reflectate.
