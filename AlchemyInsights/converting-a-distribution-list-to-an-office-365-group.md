# <a name="convert-a-distribution-group-to-a-microsoft-365-group"></a>Conversia unui grup de distribuire într-un grup Microsoft 365

Aveți posibilitatea să efectuați conversia grupurilor de distribuire la Microsoft 365 Groups dacă acestea îndeplinesc anumite criterii. Pentru a efectua conversia unui grup sau a depana problemele de conversie, consultați [Upgrade liste de distribuire la Microsoft 365 Grupuri în Outlook](https://docs.microsoft.com/microsoft-365/admin/manage/upgrade-distribution-lists).
