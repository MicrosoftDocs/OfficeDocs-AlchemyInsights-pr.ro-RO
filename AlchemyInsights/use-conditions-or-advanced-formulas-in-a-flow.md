# <a name="use-conditions-or-advanced-formulas-in-a-flow"></a>Utilizarea condițiilor sau a formulelor complexe într-un flux

În Microsoft Flow, puteți să particularizați un flux, astfel încât să efectueze activități numai dacă o condiție este adevărată. Pentru informații, consultați [adăugarea unei condiții la un flux](https://go.microsoft.com/fwlink/?linkid=872112).
  
Pentru a crea o formulă complexă, consultați [schema lingvistică pentru definirea fluxului de lucru pentru aplicațiile Azure logice](https://aka.ms/logicexpressions).
