# <a name="windows-defender-actions-in-intune"></a>Acțiuni Windows Defender în Intune

Intune poate fi utilizat pentru a declanșa o scanare la cerere și o actualizare de semnătură a virușilor pentru Windows Defender pe dispozitive individuale.

După ce o acțiune la distanță este declanșată cu succes activitatea se reflectă în jurnalul de evenimente Windows Defender.

Politica de protecție a punctelor finale Windows permite crearea de setări suplimentare pentru caracteristicile Windows Defender în Intune și aplicate seturilor de dispozitive.

Pentru mai multe detalii despre declanșarea acțiunilor Windows Defender, consultați [Configurarea și executarea scanărilor antivirus Microsoft Defender la cerere](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/run-scan-windows-defender-antivirus).
