# <a name="use-your-browser-to-check-spelling"></a>Utilizaţi browser-ul pentru verificarea ortografică

Deşi Outlook.com nu are o vraja, verificarea funciiei, multe browsere web verifica automat ortografia, inclusiv Microsoft Edge, Internet Explorer 10 şi versiunile ulterioare și versiunile curente de Firefox, Chrome si Safari. Examinaţi opţiunile pentru browser-ul web pentru a afla mai multe despre cum pentru a verifica ortografia.
