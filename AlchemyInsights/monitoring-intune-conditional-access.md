# <a name="troubleshoot-issues-with-microsoft-intune"></a>Depanarea problemelor cu Microsoft Intune

Revizuiți resursele listate mai jos pentru a rezolva problema acum.
  
1. Utilizați [portalul de depanare Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) pentru a diagnostica și a rezolva unele erori comune. Revizuiți [acest document ](https://docs.microsoft.com/intune/help-desk-operators)pentru mai multe detalii.
    
2. Revizuiți [Intune TechNet ](https://social.technet.microsoft.com/forums/home?forum=microsoftintuneprod)pentru a găsi răspunsuri și soluții la probleme comune.
    
3. Extragerea stării [serviciului](https://portal.office.com/AdminPortal/Home#/servicehealth) pentru a vedea starea serviciului. 
    
4. Revizuiți [documentația Intune](https://docs.microsoft.com/intune/) pentru planificarea, instrucțiuni și alte articole de depanare.
