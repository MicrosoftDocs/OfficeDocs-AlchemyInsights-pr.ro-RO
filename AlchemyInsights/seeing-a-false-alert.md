# <a name="seeing-a-false-alert"></a>Vizualizarea unei avertizări false

- Remiteți un fișier la Microsoft pentru analiză. Pentru mai multe informații, consultați [Ce se poate face cu rezultate fals pozitive/negative în Microsoft Defender Antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).
- Creați un indicator "Se permite" pentru a preveni o alertă fals pozitivă. Pentru mai multe informații, consultați [Crearea indicatorilor pentru fișiere](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/indicator-file).  
- Configurați excluderile de pe dispozitiv pentru a împiedica scanarea unui element. Pentru mai multe informații, consultați [Definirea unei excluderi pe un dispozitiv Windows individual pentru a împiedica scanarea unui element](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
