# <a name="self-service-password-reset-sspr"></a>Resetarea parolei cu autoservire (SSPR)

- Faceți clic aici pentru a [înregistra informațiile dvs. de contact SSPR](https://go.microsoft.com/fwlink/?linkid=849451).
