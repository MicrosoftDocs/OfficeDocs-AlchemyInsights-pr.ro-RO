# <a name="block-pop-up-windows-in-microsoft-edge"></a>Blocarea ferestrelor pop-up în Microsoft Edge

1. Accesați **... (Setări și altele)**  >  **Setări**  >  **Permisiunile site-ului**.
2. Selectați **ferestre pop-up și redirecționări**.
3. Activați **blocarea**.
