# <a name="lock-and-unlock-sites"></a>Blocarea și deblocarea site-urilor

Ca administrator Microsoft 365 global sau SharePoint, puteți să blocați accesul la un site sau să faceți un site doar în citire, utilizând Microsoft PowerShell, pentru a modifica starea de blocare a site-ului. Pentru mai multe informații, consultați Blocarea [și deblocarea site-urilor.](/sharepoint/manage-lock-status)
