# <a name="stalleddueto-errors-during-migration"></a>Erori dueTo blocate în timpul migrării

Este posibil să vedeți o varietate de erori care încep cu "StalledDueTo..." în timpul migrării la Exchange Online. Multe dintre aceste erori sunt perfect normale. Pentru mai multe informații, consultați [Limitarea bazată pe resurse și prioritizarea în migrațiile exchange online](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).
