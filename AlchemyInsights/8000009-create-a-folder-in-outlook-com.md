# <a name="create-a-folder"></a>Creaţi un folder

1. În panoul din stânga, selectaţi **Creare Folder nou**![a crea nou pliant buton](media/d8e28612-fbdb-4d28-a4d0-14f7834cfd97.png) lângă **foldere**. 
    
2. În caseta folder nou, introduceţi un nume pentru dosar şi apăsaţi Enter.
    
Pentru mai multe informaţii, consultaţi [crearea unui folder in Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=873114).
