# <a name="use-your-browser-to-check-spelling"></a>Utilizaţi browser-ul pentru verificarea ortografică

Nu există nici o opţiune de selectare a vraja în Outlook.com. Pentru a verifica ortografia, trebuie să utilizaţi browser-ul web. Verificarea ortografiei este disponibilă în Microsoft Edge, Internet Explorer 10 şi versiunile ulterioare și versiunile curente de Firefox, Chrome si Safari. Examinaţi opţiunile pentru browser-ul web pentru a afla mai multe despre cum pentru a verifica ortografia.
