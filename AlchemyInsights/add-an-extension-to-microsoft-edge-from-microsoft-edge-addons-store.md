# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Adăugarea unei extensii la Microsoft Edge din magazinul de suplimente Microsoft Edge

Pentru a adăuga o extensie la Microsoft Edge de la Magazinul Microsoft Edge Add-ons:

- Deschideți Microsoft Edge.
- Selectați **extensiile de > > [Obțineți extensii pentru Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2136408)**.
- Selectați extensia pe care doriți să o adăugați, apoi selectați **Obțineți**.
- În caseta de dialog de confirmare, examinați cu atenție modificările pe care le poate face extinderea și selectați **Adăugare extensie**.

Apoi veți primi o confirmare că extensia a fost adăugată.
