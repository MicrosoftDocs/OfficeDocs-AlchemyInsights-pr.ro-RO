# <a name="453-too-many-recipients-as780090"></a>4.5.3 prea mulţi destinatari (AS780090)

Această eroare apare atunci când volumul de trafic de e-mail la adresa IP sursă depășește limita bazat pe reputaţia (sau lipsa de reputaţie) a adresei de IP sursă.
  
Blocarea e-mail de la adresa IP sursă va expira într-o oră. Dacă adresa IP sursă este un server de poştă electronică local care îţi aparţine, Verificaţi configuraţia conectorul de fluxul de corespondenţă. În cazul în care comportamentul continuă pentru mai mult de o oră, contactaţi asistenţa pentru a solicita o excepţie pentru adresa de IP a sursei.
