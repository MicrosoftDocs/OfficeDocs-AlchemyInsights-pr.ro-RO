# <a name="remove-a-password-from-a-word-document"></a>Eliminarea unei parole dintr-un document Word

Atâta timp cât știți parola originală, eliminarea unei parole dintr-un document este un proces simplu.

1. Deschideți documentul și introduceți parola.
2. Du-te la **dosar** > **Info** > a**proteja document** > **Encrypt cu password**.
3. Goliți parola în **caseta parolă**, apoi selectați **OK**.
