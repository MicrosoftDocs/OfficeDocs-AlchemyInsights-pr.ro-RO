# <a name="cant-download-files-in-chrome"></a>Nu pot descărca fișiere în Chrome

**Problemă** Suntem conștienți de o problemă care împiedică unii clienți să poată descărca fișiere atunci când utilizează browserul Google Chrome. 

**Soluție** Utilizați [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) sau alt browser web.
Aflați mai multe despre cum să [descărcați fișiere și atașări Outlook.com](https://support.office.com/article/8d7c1ea7-4e5f-44ce-bb6e-c5fcc92ba9ab?wt.mc_id=Office_Outlook_com_Alchemy).
