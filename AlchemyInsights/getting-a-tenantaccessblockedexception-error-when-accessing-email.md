# <a name="getting-a-tenantaccessblockedexception-error-when-accessing-email"></a>Obținerea unei erori TenantAccessBlockedException la accesarea e-mailului?

Există câteva motive diferite pentru care este posibil să primiți această eroare. De exemplu:

- Verificați dacă abonamentul este valabil pe pagina Produse **de facturare** \> [& servicii](https://portal.office.com/adminportal/home#/subscriptions) din centrul de administrare Microsoft 365.

- Verificați dacă există notificări anterioare în pagina facturi **de facturare** \> [& plăți](https://portal.office.com/adminportal/home#/billoverview) din centrul de administrare Microsoft 365.

- Dacă facturarea și abonamentele pentru contul dvs., este posibil să dureze până când modificările se propagă. Încercați din nou în câteva minute, iar dacă problema persistă, contactați-ne.
