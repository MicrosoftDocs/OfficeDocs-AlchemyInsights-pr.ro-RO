# <a name="sharepoint-files-are-missing"></a>Fișierele SharePoint lipsesc

Dacă o mulțime de fișiere OneDrive sau SharePoint sunt șterse, suprascrise, deteriorate sau infectate prin malware, oferim utilizatorilor o varietate de opțiuni pentru a ajuta la procesul de recuperare.

Pentru mai multe informații, consultați [cum se recuperează elementele lipsă, șterse sau deteriorate din SharePoint Online și OneDrive pentru Business](https://go.microsoft.com/fwlink/?linkid=2110774).
