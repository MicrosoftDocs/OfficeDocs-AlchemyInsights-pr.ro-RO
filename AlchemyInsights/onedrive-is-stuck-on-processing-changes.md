# <a name="onedrive-is-stuck-on-processing-changes"></a>OneDrive este blocat la procesarea modificărilor

Dacă OneDrive afișează "Procesarea modificărilor" pentru o lungă perioadă de timp, este posibil ca aveți un fișier online deschis, o mulțime de fișiere blocate într-o coadă sau un fișier foarte mare care se sincronizează. Read [OneDrive este blocat pe "Procesarea modificărilor"](https://support.office.com/article/onedrive-is-stuck-on-processing-changes-b386b813-9b66-4e47-8c4c-2b45533edccd) pentru pașii sugerați și mai multe informații.

Dacă aveți nevoie de mai mult ajutor, puteți [trimite un e-mail la asistența OneDrive.](https://go.microsoft.com/fwlink/p/?LinkId=528676)
