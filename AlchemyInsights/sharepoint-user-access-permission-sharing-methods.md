# <a name="sharing-and-permission-methods-to-provide-or-restrict-user-access-to-sharepoint-site"></a>Partajarea și metodele de permisiune pentru a furniza sau a restricționa accesul utilizatorilor la site-ul SharePoint

- [Partajarea unui site](https://support.office.com/article/share-a-site-958771a8-d041-4eb8-b51c-afea2eae3658)

- [Modificarea tipului de link implicit atunci când utilizatorii obțin linkuri pentru partajare](https://docs.microsoft.com/sharepoint/change-default-sharing-link)

- [Oprirea partajării fișierelor sau folderelor OneDrive sau SharePoint sau modificarea permisiunilor](https://support.office.com/article/stop-sharing-onedrive-or-sharepoint-files-or-folders-or-change-permissions-0a36470f-d7fe-40a0-bd74-0ac6c1e13323)

- [Partajarea și permisiunile](https://support.office.com/article/Sharing-and-permissions-ac85fbf1-2431-49bf-8690-f1a2b98af65f#ID0EAABAAA=Manage_permissions)

- [Activarea sau dezactivarea partajării externe pentru SharePoint Online](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off)

- [Erorile de partajare OSE în SharePoint și OneDrive](https://docs.microsoft.com/sharepoint/sharepoint-onedrive-error-message)
