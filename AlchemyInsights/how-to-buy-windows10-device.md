# <a name="how-to-buy-a-new-windows-10-device"></a>Cum se cumpără un dispozitiv Windows 10 nou

Dacă vă interesează să achiziționați un dispozitiv nou care rulează Windows 10, consultați [Doriți să obțineți Windows 10?.](https://www.microsoft.com/windows/get-windows-10) Răspunzând la câteva întrebări, veți vedea recomandări pentru cel mai bun Surface, laptop, desktop sau computer all-in-in-one pe care să vă potriviți necesitățile.
