# <a name="sharepoint-hub-sites-overview"></a>Prezentare generală a site-urilor SharePoint hub

Site-urile hub SharePoint vă ajută să întrați nevoile organizației prin conectarea și organizarea de site-uri bazate pe proiect, departament, divizare, regiune etc., facilitând:

- Descoperiți conținut legat, ar fi știri și alte activități de site-ul.


- Aplicați navigarea și branding comune în toate site-urile asociate.


- Căutare pe toate site-urile asociate.


Consultați [crearea unui site de hub în SharePoint Online](https://docs.microsoft.com/sharepoint/create-hub-site) pentru mai multe informații despre se creează un site hub din noul centru de administrare sau Microsoft PowerShell. 

Consultați [asocierea unui site SharePoint cu un site de hub](https://support.office.com/article/associate-a-sharepoint-site-with-a-hub-site-ae0009fd-af04-4d3d-917d-88edb43efc05) pentru mai multe informații despre se asociază un site-uri existente la un site de hub de la noul centru de administrare sau PowerShell.  

Consultați [Eliminați un site de hub](https://docs.microsoft.com/sharepoint/remove-hub-site) pentru mai multe informații despre se elimină un site de hub existent din noul centru de administrare sau PowerShell.
