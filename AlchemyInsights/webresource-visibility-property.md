# <a name="webresource-visibility-property"></a>Proprietatea Webresource Visibility

Trebuie să selectați opțiunea **Vizibil în mod implicit** pentru ca resursa Web să apară în formular și să fie disponibilă utilizatorilor. Pentru mai multe informații, consultați [Proprietățile resurselor Web pentru formularele de aplicații bazate pe model](https://docs.microsoft.com/powerapps/maker/model-driven-apps/web-resource-properties-legacy).
