# <a name="what-is-microsoft-search-in-bing"></a>Ce caută Microsoft în Bing?

[Căutarea Microsoft în Bing](https://docs.microsoft.com/deployoffice/microsoft-search-bing#what-is-microsoft-search-in-bing) este o caracteristică care vă ajută să căutați în întreaga organizație și pe web. Utilizați bara de adrese Bing și caseta de căutare pentru a găsi documente, site-uri interne și externe, instrumente de afaceri, persoane și diagrame organizaționale, fișiere, site-uri SharePoint, conținut OneDrive, teams și conversații Yammer și multe alte resurse.
