# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable la stabilirea acreditări

Când încercarea de a seta acreditările pentru o sursă de date, puteţi întâmpina GatewayNotReachable. Acest lucru ar putea fi rezultatul unui gateway depăşite. Instalaţi ultimele gateway-ul și încercați din nou.

Pentru mai multe informaţii:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
