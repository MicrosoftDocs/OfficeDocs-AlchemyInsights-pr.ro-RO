# <a name="disable-network-scan"></a>Dezactivarea scanării rețelei

Scanările de partajare a rețelei pot influența performanța.  Pentru a vă asigura că clientul nu scanează implicit partajările de rețea/fișierele, configurați următoarele setări în aplicația Windows Defender la **True**:

- DisableScanningMappedNetworkDrivesForFullScan
- DisableScanningNetworkFiles
