# <a name="plan-your-passwordless-deployment"></a>Planificați implementarea fără parolă

Faceți upgrade la o conectare alternativă utilizând o metodă de autentificare fără parolă. Verificarea pe două factori, conectarea prin telefon sau generatorii de cod pot contribui la creșterea securității și pot face mai dificilă întreruperea contului de către alte persoane. 

Alegeți dintre Windows Hello pentru business, aplicația Microsoft Authentication sau tastele de securitate FIDO2. Pentru a determina ce metodă este cea mai bună și cum să le implementați pe dispozitivul dvs., utilizați [Ghidul de implementare Fără parolă.](https://admin.microsoft.com/adminportal/home?#/modernonboarding/passwordlesssetup)
