# <a name="control-creation-of-office-365-groups"></a>Crearea de control a grupurilor Office 365

În mod implicit, toți utilizatorii din cadrul companiei pot crea grupuri. Dacă este necesar, aveți posibilitatea să modificați acest lucru și să atribuiți dreptul de a crea grupuri unui set de persoane din cadrul companiei dvs. Rețineți că acest lucru va avea impact asupra creării de grupuri din orice serviciu Microsoft 365 (cum ar fi Outlook, Teams, Planner și așa mai departe).
  
Instrucțiuni: [Gestionarea persoanelor care pot crea grupuri Office 365](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
