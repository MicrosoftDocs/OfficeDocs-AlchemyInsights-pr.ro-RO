# <a name="update-my-profile-information-or-change-my-picture"></a>Actualizarea informațiilor de profil sau modificarea imaginii

În prezent, nu este posibil să editați și să actualizați informațiile de profil direct din Outlook pentru iOS, dar este posibil să editați aceste informații în altă parte. 

Puteți să efectuați actualizări la profil în Outlook pentru Windows sau Outlook pe web (OWA). 

**Outlook pentru Windows**: 

- **Fișier**  >  **Cont Office**  >  **Informații utilizator**  >  **Despre mine** (vă aduce la Microsoft 365 profile online) > **Actualizare profil** sau faceți clic pe pictograma Cameră pentru a vă schimba fotografia  
  
**Outlook pe web (OWA)**: 

- **Imagine**  >  de profil **Profilul meu**  >  **Actualizarea profilului**
