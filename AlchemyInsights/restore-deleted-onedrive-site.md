# <a name="restore-a-deleted-onedrive-site"></a>Restaurarea unui site OneDrive șters

Când ștergeți un utilizator în centrul de administrare Microsoft 365 (sau atunci când un utilizator este eliminat prin sincronizare Active Directory), OneDrive utilizatorului va fi reținut pentru numărul de zile pe care le specificați în centrul de administrare OneDrive. (Pentru informații, consultați [Setarea retenției implicite de fișiere pentru utilizatorii OneDrive ștergeți](https://docs.microsoft.com/onedrive/set-retention).)

* [Restaurarea unui site OneDrive șters](https://docs.microsoft.com/onedrive/restore-deleted-onedrive)

Dacă multe dintre fișierele OneDrive sunt șterse, suprascrise, corupte sau infectate cu malware, aveți posibilitatea să restaurați întregul OneDrive la o oră anterioară.

* [Restaurarea OneDrive-ului](https://support.office.com/article/Restore-your-OneDrive-fa231298-759d-41cf-bcd0-25ac53eb8a15)
