# <a name="sharepoint-hub-site-navigation"></a>Navigare site hub SharePoint

În partea de sus a unui site hub și orice site-uri asociate, deasupra navigării site-ului, este bara de navigare site-ul hub. Dacă sunteți proprietarul site-ului hub, aveți posibilitatea să particularizați [bara de navigare](https://support.office.com/article/customize-the-navigation-on-your-sharepoint-site-3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca#hubnav). 

**Notă:** Se recomandă selectarea unui site de comunicare sau a unui site de echipă utilizând șablonul modern ca site-ul hub-ului. Dacă utilizați un site clasic de echipă, navigarea hub va apărea numai pe paginile moderne, iar setările site-ului hub vor apărea doar pe pagina modernă.
