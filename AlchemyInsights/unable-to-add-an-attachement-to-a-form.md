# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Imposibil de adăugat o atașare la un formular orientat spre exterior

Dacă încercați să adăugați o atașare la un chestionar Microsoft Forms extern, acea caracteristică nu este încă disponibilă. Pentru a urmări starea solicitării de caracteristică, accesați [această intrare UserVoice](https://go.microsoft.com/fwlink/?linkid=2133069)și adăugați votul.
