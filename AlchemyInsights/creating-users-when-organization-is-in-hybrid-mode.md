# <a name="create-users-in-hybrid-deployments"></a>Creaţi utilizatori în implementări hibride

Atunci când organizaţia dumneavoastră este coexistenţei dintre un mediu local și Office 365, aveţi nevoie pentru a crea toate obiectele utilizator în Active Directory local. Creaţi obiecte utilizator în unitatea corespunzătoare organizaţionale (OU), astfel încât acestea sunt sincronizate în organizaţia Office 365.
