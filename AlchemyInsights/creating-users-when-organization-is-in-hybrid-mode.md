# <a name="create-users-in-hybrid-deployments"></a>Crearea utilizatorilor în implementări hibride

Atunci când organizația este o implementare hibridă între un mediu local și Microsoft 365, trebuie să creați toate obiectele de utilizator în Active Directory local. Creați obiectele de utilizator în unitatea organizațională corespunzătoare (OU), astfel încât acestea să fie sincronizate în organizația dvs.
