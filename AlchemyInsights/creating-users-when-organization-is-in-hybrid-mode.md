# <a name="create-users-in-hybrid-deployments"></a>Crearea utilizatorilor în implementări hibride

Atunci când organizația este o implementare hibridă între un mediu local și Microsoft 365, trebuie să creați toate obiectele de utilizator în Active Directory local. Creați obiectede utilizator în unitatea organizațională corespunzătoare (OU), astfel încât acestea sunt sincronizate în organizația dvs.
