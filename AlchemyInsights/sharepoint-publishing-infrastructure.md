# <a name="sharepoint-publishing-infrastructure"></a>SharePoint publicarea infrastructurii


SharePoint include un set clasic de caracteristici care vă permit să autor şi publica pagini web bogat pentru intranet. Activa SharePoint publicare infrastructurii la nivelul colecției de site-ul pentru a le activa.

Pentru mai multe informaţii vă rugăm să vizitaţi link-urile de mai jos.

[Activa funcţiile de publicare](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Publicare caracteristici](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Note:

- Editura clasic site-ul template-uri includ caracteristici de publicare în mod implicit.

- Caracteristicile clasice de publicare nu sunt disponibile pe site-uri moderne. Site-uri moderne oferă o alternativă nouă în mod implicit.
