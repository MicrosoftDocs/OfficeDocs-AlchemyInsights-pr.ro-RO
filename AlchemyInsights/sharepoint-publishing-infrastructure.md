# <a name="sharepoint-publishing-infrastructure"></a>Infrastructură de publicare SharePoint

> [!IMPORTANT]
> Site-urile de comunicare moderne nu acceptă caracteristicile de publicare clasice cunoscute sub numele de infrastructură de publicare SharePoint Server. Atunci când încercați să activați caracteristica pentru un site de comunicare modern, veți primi o eroare. Site-urile de comunicare moderne oferă o alternativă mai nouă în mod implicit.

Pentru site-uri clasice, SharePoint include un set clasic de caracteristici care vă permit să creați și să publicați pagini web bogate pentru intranet. Activați infrastructura de publicare SharePoint la nivel de colecție de site-uri pentru a le activa.

Pentru mai multe informații, consultați [activarea caracteristicilor de publicare](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518) și a [caracteristicilor de publicare](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)
