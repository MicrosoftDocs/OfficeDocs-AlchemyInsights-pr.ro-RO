# <a name="sharepoint-publishing-infrastructure"></a>Infrastructură de publicare SharePoint


SharePoint include un set clasic de caracteristici care vă permit să autor și să publicați pagini web bogate pentru intranet. Activați infrastructura de publicare SharePoint la nivel de colecție de site-uri pentru a le activa.

Pentru mai multe informații vă rugăm să vizitați link-urile de mai jos.

[Activați funcțiile de publicare](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Funcții de publicare](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Note:

- Șabloanele de site-uri de publicare clasice includ în mod implicit funcțiile de publicare.

- Caracteristicile clasice de publicare nu sunt disponibile pe site-urile moderne. Site-urile moderne oferă o alternativă mai nouă în mod implicit.
