# <a name="upload-process-overview-in-microsoft-stream"></a>Prezentarea generală a procesului de încărcare în Microsoft Stream

Ori de câte ori încărcați un videoclip, Microsoft Stream îl codifică automat în mai multe rate de biți, astfel încât videoclipul să poată fi redat în flux utilizând streaming adaptiv. Microsoft Stream optimizează rata de biți și calitatea videoclipului redat pe baza lățimii de bandă disponibile a spectatorului.

- [Prezentarea generală a procesului de încărcare în Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
