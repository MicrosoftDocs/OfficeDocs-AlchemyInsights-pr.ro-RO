# <a name="copy-and-paste-in-outlookcom"></a>Copierea și lipirea în Outlook.com

Nu există butoane de copiere sau lipire în Outlook.com, dar puteți utiliza comenzi rapide de la tastatură:

**Pe un PC:** CTRL + C pentru a copia, CTRL + X pentru a decupa și CTRL + V pentru a-l lipi.

**Pe un Mac:** CMD + C pentru a copia, CMD + X pentru a decupa și CMD + V pentru a-l lipi.
