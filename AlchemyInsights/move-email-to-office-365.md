# <a name="move-email-to-microsoft-365"></a>Mutarea e-mailului în Microsoft 365

Aveți mai multe opțiuni pentru migrarea e-mailului la Microsoft 365. [Consultantul de migrare a corespondenței Microsoft 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) vă ajută să alegeți cel mai bun pentru mediu și, de asemenea, să vă ghideze prin migrarea corespondenței.
  
Pentru a citi despre opțiunile dumneavoastră fără a începe migrarea, consultați [hotărâți-vă pe o cale de migrare](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

De asemenea, consultați pagina [adoptarea Microsoft 365 cu FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) pentru ajutor asistat la migrare, dacă este eligibil.
