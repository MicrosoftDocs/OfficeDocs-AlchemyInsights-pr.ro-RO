# <a name="move-email-to-microsoft-365"></a>Mutarea e-mailului la Microsoft 365

Aveți mai multe opțiuni pentru migrarea e-mail la Microsoft 365. [Consultantul microsoft pentru migrarea corespondenței 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) vă ajută să o alegeți pe cea mai bună pentru mediul dvs.
  
Pentru a citi despre opțiunile dvs., fără a începe migrarea, consultați [Decideți pe o cale de migrare](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

De asemenea, verificați pagina [Adoptați Microsoft 365 cu FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) pentru ajutor asistat în ceea ce priveste migrarea, dacă este eligibilă.
