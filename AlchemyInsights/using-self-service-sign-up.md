# <a name="using-self-service-sign-up-in-your-organization"></a>Utilizarea înscrierii cu autoservire în organizația dvs.

Cu autoservire, utilizatorii se pot înscrie pentru a utiliza serviciile plătite de abonamentul dvs.

- Pentru a controla dacă utilizatorii se pot înscrie pentru abonamente cu autoservire, utilizați cmdlet-ul [Set-MsolCompanySettings](https://docs.microsoft.com/powershell/module/msonline/set-msolcompanysettings?view=azureadps-1.0) PowerShell cu parametrul **AllowAdHocSubscriptions.** Pentru mai multe informații, consultați [Cum controlez setările de autoservire?](https://docs.microsoft.com/microsoft-365/commerce/subscriptions/self-service-purchase-faq?view=o365-worldwide)

- Pentru a vedea toate programele disponibile, consultați [Programe cu autoservire disponibile](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide#available-self-service-programs).

Pentru informații complete, consultați [Utilizarea înscrierii cu autoservire în organizația dvs.](https://docs.microsoft.com/microsoft-365/admin/misc/self-service-sign-up?view=o365-worldwide)
