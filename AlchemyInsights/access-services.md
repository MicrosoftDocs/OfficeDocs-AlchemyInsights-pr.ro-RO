# <a name="access-services"></a>Servicii de acces

Caracteristica de servicii de acces în SharePoint Online este fiind pensionat. Noul accent-bază web apps şi baze de date web nu mai pot fi create, şi aplicaţii web existente şi bazele de date web va închide până în aprilie, 2018. Acces baze de date pupitru (.accdb) nu sunt afectate de această modificare. Va recomandam Microsoft PowerApps ca o alternativă la acces web apps şi baze de date web. 
  
[Aflaţi mai multe despre Microsoft PowerApps](https://powerapps.microsoft.com/)
