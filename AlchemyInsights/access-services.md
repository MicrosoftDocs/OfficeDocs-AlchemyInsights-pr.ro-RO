# <a name="access-services"></a>Access Services

Caracteristica Access Services din SharePoint Online este retrasă. Noile aplicații web bazate pe acces și bazele de date Web nu se mai pot crea, iar aplicațiile web și bazele de date Web existente vor fi închise până în aprilie, 2018. Bazele de date desktop Access (. accdb) nu sunt afectate de această modificare. Recomandăm Microsoft PowerApps ca alternativă la accesarea aplicațiilor Web și a bazelor de date Web. 
  
[Aflați despre Microsoft PowerApps](https://powerapps.microsoft.com/)
