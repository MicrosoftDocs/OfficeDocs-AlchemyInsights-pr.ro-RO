# <a name="access-services"></a>Servicii de acces

Caracteristica Access Services din SharePoint Online este retrasă. Nu se mai pot crea aplicații web și baze de date web bazate pe Access, iar aplicațiile web și bazele de date web existente vor fi închise până în aprilie 2018. Bazele acoperire de date Desktop Access (.accdb) nu sunt afectate de această modificare. Vă recomandăm Microsoft PowerApps ca alternativă la aplicații web Access și baze de date web. 
  
[Aflați despre Microsoft PowerApps](https://powerapps.microsoft.com/)
