# <a name="microsoft-365-guest-sharing-settings-reference"></a>Referința la setările de partajare Microsoft 365 Guest

Următorul articol oferă o referință pentru diversele setări care pot afecta partajarea cu persoane din afara organizației pentru Microsoft 365 încărcări de lucru: Teams, Grupuri Office 365, SharePoint și OneDrive. Aceste setări se află în centrele de administrare Azure Active Directory, Microsoft 365, Teams și SharePoint.
