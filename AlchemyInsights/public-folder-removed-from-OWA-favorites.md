# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Folder public eliminat din Outlook de pe web (OWA)

Există o problemă cunoscută cu folderele publice care sunt eliminate din preferințele din OWA la fiecare conectare sau reîmprospătare a ecranului. Problema va fi remediată în scurt timp. Până atunci, adăugați din nou folderul pentru acces din OWA.
