# <a name="view-your-users-microsoft-365-apps-activity"></a>Vizualizarea activității aplicațiilor Microsoft 365 ale utilizatorilor

[Raportul de activare Office](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) vă oferă o vizualizare a utilizatorilor care și-au instalat și activat aplicațiile Microsoft 365 pe cel puțin un dispozitiv.

În plus, puteți să obțineți o vizualizare în activitatea de aplicații Microsoft 365 a utilizatorilor, utilizând [raportul de utilizare a aplicațiilor microsoft 365](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide).
