# <a name="join-a-team-by-code"></a>Asocierea la un cod de echipă

Microsoft cunoaște o problemă în care utilizatorii nu pot să creeze un cod de **echipă** sau să se asocieze la o echipă utilizând caracteristica **asociere după cod** . Ca soluție de evitare, proprietarii echipei pot adăuga manual utilizatori, după cum este necesar.
