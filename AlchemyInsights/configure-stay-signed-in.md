# <a name="configure-stay-signed-in-for-accounts"></a>Configurarea opțiunii Rămâneți conectat pentru conturi

Păstrează-mă conectat (KMSI) afișează o solicitare **Rămâneți conectat?** după ce un utilizator se conectat cu succes. Dacă un utilizator răspunde **da** la această solicitare, serviciul KMSI îi oferă un simbol de reîmprospătare persistentă. 

Pentru mai multe informații despre configurarea KMSI, consultați Configurarea solicitării ["Rămâneți conectat?" pentru conturile Azure AD.](/azure/active-directory/fundamentals/keep-me-signed-in)
