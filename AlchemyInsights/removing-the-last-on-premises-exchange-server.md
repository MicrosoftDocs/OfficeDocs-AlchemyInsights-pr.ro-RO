# <a name="removing-the-last-on-premises-exchange-server"></a>Scoate Ultima serverul Exchange local

Înainte de a scoate Ultima Exchange server din organizaţia locală, trebuie să asiguraţi-vă că înţelegeţi implicaţiile de a face acest lucru pentru a vă ajuta să decideţi dacă aveţi nevoie de acel server Ultima pentru orice alte scopuri. Pentru mai multe informaţii, a verifica afară [cum şi când să cum dezactivez serverele Exchange local coexistenţei](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
