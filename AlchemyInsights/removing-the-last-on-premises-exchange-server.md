# <a name="removing-the-last-on-premises-exchange-server"></a>Eliminarea ultimului server Exchange local

Înainte de a elimina ultimul server Exchange din organizația locală, trebuie să vă asigurați că înțelegeți implicațiile de a face acest lucru pentru a vă ajuta să decideți dacă aveți nevoie de ultimul server în orice alt scop. Pentru mai multe informații, consultați [cum și când să vă deconectați serverele Exchange locale dintr-o implementare hibridă](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
