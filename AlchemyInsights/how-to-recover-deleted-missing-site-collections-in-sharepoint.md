# <a name="recover-missing-or-deleted-site-collections"></a>Recuperați colecții de site-uri lipsă sau șterse

Consultați următoarele articole pentru informații despre să recuperați colecții de site-uri lipsă sau șterse în SharePoint.

[Restaurare site-uri șterse](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Restaurare utilizând modulul SPO PowerShell](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Dacă nu găsiți site-ul poate doriți să [Căutați Jurnalul de audit în Office 365 &amp; Security conformitatea Center](https://docs.microsoft.com/office365/securitycompliance/search-the-audit-log-in-security-and-compliance).

**Notă:** "Modern" echipa site-uri nu sunt vizibile în **clasic** SharePoint admin UI. Puteți accesa lista de site-uri "moderne" din [noul centru de administrare SharePoint](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
