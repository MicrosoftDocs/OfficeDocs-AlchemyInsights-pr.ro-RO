# <a name="download-files-and-folders-from-onedrive-or-sharepoint"></a>Descărcarea de fișiere și foldere din OneDrive sau SharePoint

Puteți să descărcați fișiere și foldere din Microsoft OneDrive sau din SharePoint în Microsoft 365 sau SharePoint Server 2019, pe computer cu doar câteva clicuri. Pentru mai multe informații, consultați [descărcarea fișierelor și folderelor din OneDrive sau SharePoint](https://support.microsoft.com/office/download-files-and-folders-from-onedrive-or-sharepoint-5c7397b7-19c7-4893-84fe-d02e8fa5df05).
