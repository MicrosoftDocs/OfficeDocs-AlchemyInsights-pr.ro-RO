# <a name="sharepoint-online-public-websites-have-been-discontinued"></a>Site-urile web publice SharePoint Online au fost întrerupte

Site-urile web publice SharePoint Online au fost întrerupte.

Dacă ați creat un site Web SharePoint Online public, ați reușit să continuați să-l utilizați prin 31 martie 2018. Dacă doriți să continuați să aveți un site web public, trebuie să creați unul nou cu un nou furnizor. Conținutul de pe site-ul web public SharePoint Online a fost șters.

Pentru informații despre întreruperea site-ului public, consultați [Detalii despre procesul de întrerupere a caracteristicii](https://go.microsoft.com/fwlink/?linkid=866980).
