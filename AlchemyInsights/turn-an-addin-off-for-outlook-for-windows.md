# <a name="turn-an-add-in-off-for-outlook-for-windows"></a>Activarea unui program de completare pentru Outlook pentru Windows

Dacă nu doriți să utilizați un program de completare în Outlook 2013 sau Outlook 2016 pentru Windows, puteți să-l dezactivați, astfel încât să nu se afișeze în mesaje.  

1. În Outlook, faceți clic pe **fișier > Gestionați programele de completare**.
2. Faceți clic pe  **Gestionați programele de completare**
3. Sfat: Aceasta deschide Outlook pe web.
4. Sub Gestionați programele de completare, în coloana activat, **debifați caseta pentru programul de completare**  pe care doriți să-l dezactivați.
5. Activarea sau dezactivarea programelor de completare

Sfat: pentru a activa din nou programul de completare, bifați aceeași casetă.
