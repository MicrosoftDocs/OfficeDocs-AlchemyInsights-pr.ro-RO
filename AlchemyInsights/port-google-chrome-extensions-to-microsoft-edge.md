# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Portați extensii Google Chrome la Microsoft Edge (crom)

Este ușor să [portați extensii Google Chrome la Microsoft Edge (crom)](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension). În majoritatea cazurilor, sunt necesare doar modificări minime pentru a executa aceste extensii în Microsoft Edge.

API-urile de extensie și tastele manifest acceptate de Google Chrome sunt compatibile cu codul cu Microsoft Edge. Cu toate acestea, Microsoft Edge nu acceptă extensia APIs Chrome. GCM, Chrome. Identity. getAccounts, Chrome. Identity. getAuthToken și Chrome. instanceID.
