# <a name="port-google-chrome-extensions-to-microsoft-edge-chromium"></a>Portați extensiile Google Chrome la Microsoft Edge (Chromium)

Este simplu să [portați extensiile Google Chrome în Microsoft Edge (Chromium).](https://docs.microsoft.com/microsoft-edge/extensions-chromium/developer-guide/port-chrome-extension) În majoritatea cazurilor, sunt necesare doar modificări minime pentru a rula aceste extensii în Microsoft Edge.

API-urile extensie și cheile manifeste acceptate de Google Chrome sunt compatibile cu codul cu Microsoft Edge. Cu toate acestea, Microsoft Edge nu acceptă extensia APIs chrome.gcm, chrome.identity.getAccounts, chrome.identity.getAuthToken și chrome.instanceID.
