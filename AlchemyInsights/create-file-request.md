# <a name="how-to-create-a-file-request"></a>Cum se creează o solicitare de fișier

Caracteristica de solicitare a fișierelor din OneDrive vă permite să alegeți un folder în care alte persoane să poată încărca fișiere utilizând un link pe care îl trimiteți. Utilizatorii la care solicitați fișiere nu pot vedea ce se află în folder-pot încărca doar fișiere în acesta.

**Notă**: un administrator trebuie să activeze [orice linkuri](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) din OneDrive pentru ca solicitările de fișier să funcționeze.

- Pentru mai multe informații, consultați [crearea unei solicitări de fișier](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af).
