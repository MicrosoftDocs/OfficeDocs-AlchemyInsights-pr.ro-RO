# <a name="how-to-create-a-file-request"></a>se creează o solicitare de fișier

Caracteristica de solicitare a fișierelor din OneDrive vă permite să alegeți un folder în care alte persoane să poată încărca fișiere utilizând un link pe care îl trimiteți. Utilizatorii de la care solicitați fișiere nu pot vedea ce se află în folder- pot încărca doar fișiere pe el.

**Notă:** Un administrator trebuie să activeze [oricine link-uri](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) în OneDrive pentru solicitările de fișiere să funcționeze.

- Pentru mai multe informații, consultați [Crearea unei solicitări de fișier](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af).
