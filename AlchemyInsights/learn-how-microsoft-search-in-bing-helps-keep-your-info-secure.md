# <a name="learn-how-microsoft-search-in-bing-helps-keep-your-information-secure"></a>Aflați cum caută Microsoft în Bing vă ajută să păstrați informațiile securizate

Cu Microsoft Search în Bing:

- Doar utilizatorii conectați pot vedea rezultatele **căutării** .
- Identitatea de lucru sau identitatea organizației nu este utilizată de Microsoft pentru a direcționa publicitate.
- Administratorul de căutare poate vedea doar numărul de căutări după tip, de exemplu după persoană sau tip de fișier și o listă agregată de căutări de top.
- Numai dumneavoastră puteți vedea istoricul **căutărilor** la locul de muncă.
- Puteți găsi doar fișierele de lucru și informațiile care sunt partajate cu dumneavoastră sau pe care aveți permisiunea de a le vedea.
- Informațiile dumneavoastră nu sunt disponibile niciodată altor organizații, inclusiv Microsoft sau căutărilor publice efectuate pe Bing.

Pentru mai multe informații, consultați [modul în care Microsoft Search în Bing vă ajută să păstrați informațiile securizate](https://support.microsoft.com/office/how-microsoft-search-in-bing-helps-keep-your-info-secure-cbce46ae-bb1f-4d0e-86f1-5984f4589113).
