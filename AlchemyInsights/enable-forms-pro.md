# <a name="enable-or-disable-forms-pro"></a>Activarea sau dezactivarea Forms Pro

Dacă încercați să activați Microsoft Forms Pro, acesta este inclus ca parte a licenței [Dynamics 365 enterprise.](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant) Alternativ, un administrator poate achiziționa o [licență Microsoft Forms Pro](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant). De asemenea, aveți posibilitatea să [eliminați o licență Microsoft Forms Pro](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1) de la un utilizator.
