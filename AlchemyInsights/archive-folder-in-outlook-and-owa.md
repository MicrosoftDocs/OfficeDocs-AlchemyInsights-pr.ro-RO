# <a name="archive-folder-in-outlook-and-owa"></a>Arhivați folderul în Outlook și OWA

Arhiva este un folder implicit în care mesajele de e-mail pot fi depuse și organizate în cutia poștală. Aveți posibilitatea să mutați elemente în acest folder făcând clic pe butonul **Arhivă.** Elementele sunt mutate (nu șterse) în folderul **Arhivă** din aceeași cutie poștală principală și nu în cutia poștală de arhivă.

Pentru a vizualiza folderul **Arhivă,** asigurați-vă că lista de foldere este afișată selectând **Vizualizare**  >  **panou foldere**, apoi selectând **Normal**.

**Notă** Dacă elementele se deplasează automat în folderul Arhivă Outlook în mod neașteptat, este posibil să fi utilizat cheia backspace în Outlook sau să fi făcut clic pe opțiunea **Arhivare** din Outlook Desktop, Outlook Web App sau Outlook Mobile App.

Pentru mai multe informații despre folderul Arhivă din Outlook, consultați [Folderul Arhivă din Outlook](https://support.office.com/article/archive-in-outlook-for-windows-25f75777-3cdc-4c77-9783-5929c7b47028)sau pentru OWA, consultați Folderul Arhivă în [OWA](https://support.office.com/article/organize-your-inbox-with-archive-sweep-and-other-tools-in-outlook-on-the-web-49b26f63-6399-4b4a-a580-14b9b1efe96d?ui=en-US&rs=en-US&ad=US).
