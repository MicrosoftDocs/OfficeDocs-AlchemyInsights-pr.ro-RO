# <a name="learn-how-microsoft-edge-supports-windows-information-protection-wip"></a>Aflați cum acceptă Microsoft Edge protecția informațiilor Windows (PN)

Deoarece numărul de lucrători la distanță crește, la fel și numărul de activități personale și de lucru care apar pe dispozitivele corporative. Acest lucru, la rândul său, mărește riscul ca datele corporative să fie partajate în afara locului de muncă. Pentru a atenua acest risc, Microsoft Edge (versiunea 81 sau mai recentă) acceptă Windows Information Protection (PN): o caracteristică Windows-10 care ajută la protejarea datelor de întreprindere împotriva divulgării neautorizate și accidentale.
