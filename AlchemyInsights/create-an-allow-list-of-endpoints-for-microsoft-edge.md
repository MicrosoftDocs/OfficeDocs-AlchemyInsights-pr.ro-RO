# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Crearea unei liste permise de puncte finale pentru Microsoft Edge

Pentru a avea o gamă completă de caracteristici Microsoft Edge la dispoziție, trebuie să îl aveți conectat la internet. În plus, pentru a permite browserului să comunice prin paravane de protecție și alte mecanisme de securitate pe Internet, trebuie să adăugați un set de URL-uri de domeniu la [lista sa permisă de puncte finale](https://go.microsoft.com/fwlink/?linkid=2135054).
