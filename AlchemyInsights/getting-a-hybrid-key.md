# <a name="getting-an-exchange-hybrid-key"></a>Noţiuni de bază un hibrid de schimb cheie

Pentru a vă conecta organizaţia locală Exchange Online în Office 365, trebuie să setaţi până cel puţin un server Exchange în cadrul organizaţiei locale. Acest server este folosit pentru a crea o implementare hibridă, care conectează organizaţia locală Exchange în Exchange Online. Pentru a obţine o cheie de hibrid pentru acest server, urmaţi paşii de la [schimb hibrid rezultat cheie de distribuţie Expertul](https://aka.ms/hybridkey).
