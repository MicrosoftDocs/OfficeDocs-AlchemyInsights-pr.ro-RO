# <a name="getting-an-exchange-hybrid-key"></a>Obținerea unei chei hibridexchange

Pentru a conecta organizația locală la Exchange Online în Microsoft 365, trebuie să configurați cel puțin un server Exchange în organizația locală. Acest server este utilizat pentru a crea o implementare hibridă care conectează organizația Exchange locală la Exchange Online. Pentru a obține o cheie hibridă pentru acest server, urmați pașii de la [Expertul de distribuire a cheii de produs hibrid Exchange](https://aka.ms/hybridkey).
