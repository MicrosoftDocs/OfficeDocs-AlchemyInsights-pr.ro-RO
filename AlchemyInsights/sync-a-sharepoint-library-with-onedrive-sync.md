# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>Sincronizarea unei biblioteci SharePoint cu Sincronizarea OneDrive

Pentru ca fișierele SharePoint să se sincronizeze cu unitatea de hard disk locală, parcurgeți următorii pași:

1. Navigați la biblioteca de documente pe care doriți să o sincronizați cu unitatea de hard disk locală.
2. Selectați butonul Sincronizare din opțiunile de meniu.
3. Urmați pas cu pas pentru a finaliza sincronizarea.
