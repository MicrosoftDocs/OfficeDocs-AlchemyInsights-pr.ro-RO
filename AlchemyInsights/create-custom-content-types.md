# <a name="create-custom-content-types"></a>Crearea tipurilor de conținut particularizate

1. Selectați lista cu care doriți să lucrați în lansare rapidă sau faceți clic pe **Setări**  (pictograma roată), apoi faceți clic pe  **Setări site**.
2. Faceți clic pe tipuri de **conținut de site**  sub  **galerii web designer**.
3. Pe pagina tipuri de conținut de site, faceți clic pe Creare.
4. Pe pagina tip de conținut site nou, completați informațiile pentru tipul de conținut particularizat.

Pentru mai multe informații, consultați  [tipuri de conținut particularizate](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
