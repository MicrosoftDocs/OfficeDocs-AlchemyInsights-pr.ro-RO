# <a name="credential-messages-in-sharepoint-or-onedrive"></a>Mesaje de acreditare în SharePoint sau OneDrive

<p>Utilizatorii pot fi periodic semnalare de biroul acreditări SharePoint Online sau OneDrive atunci când încearcă să deschidă documente.</p> <p>Următoarele sunt câteva exemple de mesaje de acreditare:</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Faceţi sign in cu contul organizatorice</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Acreditările sunt necesare</p> <p>&bull;<span style="mso-tab-count: 1;"> &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; </span>Suntem în imposibilitatea de a conecta chiar acum, verificaţi conexiunea la reţea.</p> <p>Dacă vă confruntaţi cu aceste simptome, vă rugăm să încercaţi paşii în <a href="https://support.microsoft.com/en-us/help/2913639/office-applications-periodically-prompt-for-credentials-to-sharepoint">acest articol</a>.</p> <p>Alternativ, dacă utilizaţi soluţia de sign-on (unic SSO) pentru a autentifica, vă rugăm să consultaţi <a href="https://support.microsoft.com/en-us/help/4025962/cant-sign-in-after-update-to-office-2016-build-16-0-7967-on-windows-10">acest articol</a> în schimb.</p> <p>&nbsp;</p>
